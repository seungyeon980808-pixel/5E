# 5E를 말로 시켜서 그리기 — 설치 안내

5E(https://seungyeon980808-pixel.github.io/5E/)를 열어 두고, Claude에게 "회로도
그려줘" 같은 말만 하면 화면에 바로 그려지는 기능입니다. **정확히는 "5E의 기능"이
아니라 "여러분의 Claude 사용권 위에 붙이는 개인 도구"** 라서, 이 기능을 쓰려면
아래 준비가 각자 필요합니다.

## 필요한 것

- **Claude 유료 구독** (Claude Code를 쓸 수 있는 플랜)
- **Node.js** — 컴퓨터에 설치된 적 없다면 아래 1단계에서 설치
- 인터넷 브라우저 (Chrome, Edge 등)

설치는 한 번만 하면 됩니다.

---

## 1단계 — Node.js 설치 (이미 있다면 건너뛰기)

1. https://nodejs.org 접속
2. **LTS**라고 쓰인 초록 버튼 클릭 → 다운로드된 설치 파일 실행
3. 계속 "다음"만 눌러서 설치 끝

확인: 컴퓨터에서 "명령 프롬프트"(Windows) 또는 "터미널"(Mac)을 열어 아래를 입력합니다.

```bash
node -v
```

버전 번호(`v20.x.x` 같은 것)가 나오면 성공입니다.

## 2단계 — 도구 내려받기

1. https://github.com/seungyeon980808-pixel/5E 접속
2. 초록색 **Code** 버튼 → **Download ZIP**
3. 압축을 원하는 폴더에 풉니다 (예: 바탕화면 `5E` 폴더)

## 3단계 — Claude Code에 등록

명령 프롬프트/터미널에서, 압축을 푼 폴더 경로를 넣어 아래를 실행합니다.
(경로의 `5E`는 압축 풀린 실제 폴더 이름으로 바꾸세요 — 보통 `5E-main`으로 풀립니다)

```bash
claude mcp add 5e -- node "여기에_압축푼_경로/tools/mcp-5e/server.js"
```

예시 (Windows, 바탕화면에 풀었을 때):

```bash
claude mcp add 5e -- node "C:/Users/내이름/Desktop/5E-main/tools/mcp-5e/server.js"
```

성공하면 `Added stdio MCP server 5e...` 라는 메시지가 뜹니다.

## 4단계 — Claude Code를 새로 켜기

**중요**: 등록한 직후의 창(터미널)이 아니라, **새 터미널 창**에서 `claude`를 다시
실행해야 방금 등록한 도구가 인식됩니다.

## 5단계 — 5E를 열고 연결 확인

브라우저에서 아래 주소로 접속합니다 (`?mcp=1`이 핵심입니다 — 이 브라우저에 한 번만
켜두면 됩니다).

```
https://seungyeon980808-pixel.github.io/5E/?mcp=1
```

화면 위쪽, zoom 표시 왼쪽에 있는 **MCP ●** 을 확인합니다.

- **파란 점**: 연결됨 — 바로 쓰시면 됩니다
- **회색 점**: 클릭 한 번 (Claude Code가 이미 켜져 있으면 바로 파란색으로 바뀝니다)

## 이제부터 매번 쓰는 법

1. Claude Code를 켠다 (`claude`)
2. 5E를 브라우저에서 연다 (즐겨찾기에 `?mcp=1` 붙은 주소를 저장해 두면 편합니다)
3. 편하게 말한다 — "9V 전지에 저항 2개 직렬 회로 그려줘" 같은 식으로
4. 마음에 안 들면 **Ctrl+Z**

---

## 자주 막히는 지점

| 증상 | 원인 / 해결 |
|---|---|
| `node -v`가 안 먹힘 | Node.js 설치 후 터미널을 새로 열지 않아서인 경우가 많습니다 |
| `claude mcp add` 후에도 안 됨 | 등록한 그 창이 아니라 **새 창**에서 Claude Code를 켰는지 확인 |
| MCP 점이 계속 회색 | ① Claude Code가 켜져 있는지 ② 5E가 `?mcp=1`로 열려 있는지 ③ 점을 한 번 클릭했는지 |
| 5E를 여러 탭에서 동시에 열어둠 | 마지막에 연 탭으로만 그려집니다 — 한 탭만 열어 두세요 |
| 회사/학교 컴퓨터라 설치가 막힘 | 관리자 권한이 필요할 수 있습니다 — 개인 노트북에서 먼저 시도해 보세요 |

## 이게 왜 이렇게 복잡한가 (궁금하실 분들을 위해)

이 기능은 웹사이트가 아니라 **여러분 컴퓨터 안에서 도는 개인 프로그램**입니다.
브라우저가 보안상 여러분 컴퓨터의 프로그램을 대신 설치·실행해 줄 수 없어서,
이 몇 단계는 줄일 수가 없습니다. 대신 한 번만 해두면 그다음부터는 그냥 말로
시키기만 하면 됩니다.
