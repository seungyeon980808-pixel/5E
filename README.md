**5E**(sciEnceEducationalExamEasyEditor)는 과학 교사가 시험지·학습지에 넣을 그림을 빠르게 그리고 편집하기 위한 웹 기반 도구입니다. 설치 없이 브라우저에서 동작하며, 공통 도구로 다양한 이미지를 만들고 라이브러리에서 기출문항을 불러와 편집할 수 있습니다.

## 대상 사용자

- 과학(물리·화학·생명·지구과학) 교사
- 시험지·학습지·수업자료 그림 제작자

## 주요 기능

파워포인트나 그림판 대신, 시험지·학습지에 넣을 과학 그림을 **정확한 벡터 도형**으로 빠르게 만들도록 설계했습니다. 아래는 5E를 쓰는 핵심 이유입니다.

### 과목별 라이브러리와 기출문항 검색

5E의 출발점은 "처음부터 그리기"가 아니라 "있는 그림을 고쳐 쓰기"입니다.

- **과목 전환** — 물리·화학·생명·지구과학 중 하나를 고르면 강조색 테마와 함께 과목별 오브젝트 팔레트가 통째로 바뀝니다.
- **기출문항 검색 (`Ctrl+Shift+F`)** — 라이브러리에 담긴 기출 도해를 검색해 캔버스로 불러온 뒤, 바로 편집해 새 문항에 맞춥니다.
- **퍼스널 오브젝트** — 자주 쓰는 도해를 개인 저장소에 저장해 언제든 다시 꺼내 씁니다.

### 그래프 · 함수 도구

좌표평면 위에 함수·직선·꺾은선·곡선을 한 화면에서 만듭니다.

- **수식으로 그래프** — 수식을 입력하면 조밀 샘플링으로 매끄럽고 정확한 곡선을 그립니다. 여러 함수를 한 평면에 겹쳐 넣을 수 있고, 축·눈금은 평가원 시험지 양식을 따릅니다.
- **찍은 점을 정확히 지나는 곡선** — 손으로 찍은 점을 그대로 통과하는 centripetal Catmull-Rom 보간으로 자연스러운 곡선을 만듭니다.
- **비대칭 좌표평면** — x·y의 음/양 방향 칸 수를 따로 지정해 필요한 범위만 그립니다.

### 그리기 도구

- **선** — 직선, 꺾은선, 곡선, 자유 그리기
- **도형** — 타원, 사각형, 직각삼각형
- **텍스트·수식** — 자유 텍스트와 LaTeX 수식(팔레트 버튼으로 기호 입력), 지시선이 달린 **라벨러**
- **표시·측정** — 각도 호, 직각 표시, 점, 길이 표시, 그리고 화면 위에 올려 쓰는 **자·각도기**
- **자르기(가위)** — 자유곡선으로 오브젝트를 자르고, 채움을 유지한 채 둘로 나눕니다

### 이미지 작업

- **이미지 객체화** — 불러온 이미지를 편집 가능한 벡터 객체로 바꿔, 필요한 부분만 남기거나 고쳐 씁니다.
- **여러 이미지 배치** — 한 페이지에 여러 장을 모으거나 페이지별로 하나씩 불러오고, 위치·비율 고정과 원본 비교를 지원합니다.

### 작업 편의

- **다중 페이지** — 아트보드를 페이지 탭으로 여러 개 관리합니다(추가·복제·순서 변경).
- **전체 통일/수정** — 선택(또는 전체) 오브젝트의 굵기·색·글씨·각도·잠금을 한 번에 통일하거나 증감합니다.
- **정밀 배치** — 스냅, 격자, 눈금자, 중앙 고정으로 오차 없이 붙이고 정렬합니다.
- **자동 저장** — 작업이 브라우저에 자동 보관되어, 실수로 창을 닫아도 이어서 작업할 수 있습니다.

### 저장 · 내보내기

- **프로젝트 저장·열기** — JSON으로 저장하고, 파일을 끌어다 놓아 바로 엽니다.
- **이미지 내보내기** — PNG / JPG, 선택 영역만 지정해 내보내기
- **전체 백업/복원** — 설정 + 퍼스널 라이브러리 + 현재 프로젝트를 한 파일로 묶어 보관합니다.

### 화면 · 환경

- **Pro / Lite 모드** — 단축키에 익숙하지 않은 입문자를 위한 Lite(도구 확대·고급 기능 숨김)와, 모든 기능을 쓰는 Pro를 버튼 하나로 전환합니다.
- **환경 설정** — 글씨·도구 패널 크기(소형·중형·대형·와이드), 다크 테마 기본, 전체화면
- **모바일 대응** — 터치로 그리기·이동, 좁은 화면에서는 좌우 패널을 드로어로 여닫습니다.

## 로컬 실행

정적 웹앱이라 별도 빌드가 필요 없습니다. 저장소를 받은 폴더에서 정적 서버를 띄우면 됩니다.

```powershell
python -m http.server 8000
```

브라우저에서 열기:

```text
http://localhost:8000
```

Windows에서는 저장소에 포함된 `run-server.bat`을 더블클릭해도 됩니다.

## 저장 파일 형식

프로젝트 JSON은 `objects`, `guides`, `layers`, `artboard`를 스키마 `0.17`로 저장합니다. 되돌리기 히스토리, 선택 상태, 현재 화면 배율(viewBox)은 저장되지 않습니다. 구버전 파일은 열 때 현재 스키마로 자동 마이그레이션됩니다.
