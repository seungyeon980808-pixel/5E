**5E**(sciEnceEducationalExamEasyEditor)는 과학 교사가 시험지·학습지에 넣을 그림을 빠르게 그리고 편집하기 위한 웹 기반 도구입니다. 설치 없이 브라우저에서 동작하며, 공통 도구로 다양한 이미지를 만들고 라이브러리에서 기출문항을 불러와 편집할 수 있습니다.

## 대상 사용자

- 과학(물리·화학·생명·지구과학) 교사
- 시험지·학습지·수업자료 그림 제작자

## 주요 기능

파워포인트나 그림판 대신, 시험지·학습지에 넣을 과학 그림을 **정확한 벡터 도형**으로 빠르게 만들도록 설계했습니다. 아래는 5E를 쓰는 핵심 이유입니다.

### 과목별 라이브러리와 기출문항 검색

5E의 출발점은 "처음부터 그리기"가 아니라 "있는 그림을 고쳐 쓰기"입니다.

- **과목 전환** — 물리·화학·생명·지구과학 중 하나를 고르면 강조색 테마와 함께 과목별 오브젝트 팔레트가 통째로 바뀝니다.
- **기출문항 검색 (`Ctrl+Shift+F`)** — 라이브러리에 담긴 기출 도해를 검색해 캔버스로 불러온 뒤, 바로 편집해 새 문항에 맞춥니다. 번호(`261101` = 2026학년도 수능 1번)나 해시태그로도 찾습니다.
- **참고 창** — 고른 문항을 **별도 브라우저 창**으로 띄워 둡니다. 듀얼 모니터의 다른 화면에 놓고 원본을 보면서 작업하라고 만든 기능입니다. 문항마다 메모를 달 수 있고, 같은 문항을 다시 열면 적어 둔 메모가 그대로 나옵니다.
- **퍼스널 오브젝트** — 자주 쓰는 도해를 개인 저장소에 저장해 언제든 다시 꺼내 씁니다.

### 그래프 · 함수 도구

좌표평면 위에 함수·직선·꺾은선·곡선을 한 화면에서 만듭니다.

그래프 만들기 창은 **① 좌표 → ② 함수 → ③ 표시** 세 단계로 나뉩니다. 축을 만들고, 그 위에 함수를 얹고, 마지막으로 점과 안내선을 표시합니다.

- **수식으로 그래프** — 수식을 입력하면 조밀 샘플링으로 매끄럽고 정확한 곡선을 그립니다. 여러 함수를 한 평면에 겹쳐 넣을 수 있고, 축·눈금은 평가원 시험지 양식을 따릅니다.
- **정의역·치역 지정** — 함수가 그려질 구간을 값으로 입력하거나, 평면 위에 **사각형을 끌어 한 번에** 잡습니다. 그래프의 일부만 보여주는 문항을 만들 때 씁니다.
- **찍은 점을 정확히 지나는 곡선** — 손으로 찍은 점을 그대로 통과하는 centripetal Catmull-Rom 보간으로 자연스러운 곡선을 만듭니다. 격자에 갇히지 않고 좌표평면 끝까지 그립니다.
- **표시 요소** — 표시점 · 수선의 발 · 화살표 · 가이드 점선 · 범례 박스를 **함수 없이도** 얹습니다. 빈 좌표평면에 점만 찍어 만드는 그래프도 됩니다.

<img src="docs/images/release-v1.2.0/02-annot-after.png" alt="빈 좌표평면에 표시점·수선의 발·안내 점선·범례를 올린 모습" width="420" />

- **라벨러 표시점** — 클릭한 순서대로 A·B·C 이름표가 자동으로 붙고 점과 함께 움직입니다. 이름표 방향은 <kbd>PageUp</kbd> <kbd>PageDown</kbd>으로 15°씩 돌립니다.

<img src="docs/images/release-v1.2.0/04-labelpt-after.png" alt="점 주위를 15도씩 도는 이름표" width="420" />

- **비대칭 좌표평면** — x·y의 음/양 방향 칸 수를 따로 지정하고, 네 끝의 화살표 여백과 격자 튀어나옴도 각각 정합니다.
- **평가원 화살촉** — 좌표축과 직선 끝의 화살촉을 실제 기출 지면에서 윤곽을 따 재현했습니다.

### 그리기 도구

- **선** — 직선, 꺾은선, 곡선, 자유 그리기. 자유곡선은 **대충 끌어 그린 뒤** 앵커를 옮기거나 베지어 핸들로 곡률을 다듬는 방식입니다(앵커 개수도 늘리고 줄일 수 있습니다).
- **도형** — 타원, 사각형, 직각삼각형
- **텍스트·수식** — 자유 텍스트와 LaTeX 수식(팔레트 버튼으로 기호 입력), 지시선이 달린 **라벨러**. 시험지 지면에 맞춰 **자간·장평**까지 조절합니다.
- **표시·측정** — 각도 호, 직각 표시, 점, 길이 표시, 그리고 화면 위에 올려 쓰는 **자·각도기**
- **자르기(가위)** — 자유곡선으로 오브젝트를 자르고, 채움을 유지한 채 둘로 나눕니다

### 이미지 작업

- **이미지 객체화** — 불러온 이미지를 편집 가능한 벡터 객체로 바꿔, 필요한 부분만 남기거나 고쳐 씁니다. 인식 강도(묶음 거리·최소 크기·글자 판정·곡선 단순화)를 조절하고, 미리보기에서 원치 않는 조각을 클릭해 뺄 수 있습니다.
- **여러 이미지 배치** — 한 페이지에 여러 장을 모으거나 페이지별로 하나씩 불러오고, 위치·비율 고정과 원본 비교를 지원합니다.

### 작업 편의

- **다중 페이지** — 아트보드를 페이지 탭으로 여러 개 관리합니다(추가·복제·순서 변경).
- **전체 통일/수정** — 선택(또는 전체) 오브젝트의 굵기·색·글씨·각도·잠금을 한 번에 통일하거나 증감합니다.
- **정밀 배치** — 스냅, 격자, 눈금자, 중앙 고정으로 오차 없이 붙이고 정렬합니다.
- **자동 저장** — 작업이 브라우저에 자동 보관되어, 실수로 창을 닫아도 이어서 작업할 수 있습니다.

### 저장 · 내보내기

- **프로젝트 저장·열기** — JSON으로 저장하고, 파일을 끌어다 놓아 바로 엽니다.
- **이미지 내보내기** — PNG / JPG, 선택 영역만 지정해 내보내기
- **페이지 일괄 내보내기** — 저장 폴더를 한 번 연결하면 다음에도 기억하고, 내보낼 페이지를 골라 한 번에 저장합니다. 페이지에 지어준 이름이 그대로 파일명이 됩니다.
- **전체 백업/복원** — 설정 + 퍼스널 라이브러리 + 현재 프로젝트를 **ZIP 한 파일**로 묶어 보관합니다(이미지를 바이너리로 담아 용량이 작습니다). 예전 JSON 백업도 그대로 열립니다.
- **용량 제한 없는 라이브러리** — 퍼스널 오브젝트를 IndexedDB에 보관해, 브라우저 저장소의 5MB 상한에 걸리지 않습니다.

### 화면 · 환경

- **Pro / Lite 모드** — 단축키에 익숙하지 않은 입문자를 위한 Lite(도구 확대·고급 기능 숨김)와, 모든 기능을 쓰는 Pro를 버튼 하나로 전환합니다.
- **환경 설정** — 화면 · 편집 도구 · 저장 · 라이브러리 · 고급 **탭**으로 나뉘어 있습니다. 글씨와 도구 패널 크기는 슬라이더로 60~140% 자유롭게 맞춥니다.
- **다크 테마 기본 · 전체화면**
- **모바일 대응** — 터치로 그리기·이동, 좁은 화면에서는 좌우 패널을 드로어로 여닫습니다.

## 로컬 실행

정적 웹앱이라 별도 빌드가 필요 없습니다. 저장소를 받은 폴더에서 정적 서버를 띄우면 됩니다.

```powershell
python -m http.server 8000
```

브라우저에서 열기:

```text
http://localhost:8000
```

Windows에서는 저장소에 포함된 `run-server.bat`을 더블클릭해도 됩니다.

## 저장 파일 형식

프로젝트 JSON은 스키마 `0.17`을 쓰며, **페이지 단위**로 `objects`, `guides`, `layers`, `artboard`와 문항 메타를 담습니다(`pages[]`). 되돌리기 히스토리, 선택 상태, 현재 화면 배율(viewBox)은 저장되지 않습니다. 구버전 파일은 열 때 자동 마이그레이션됩니다 — 페이지 개념이 없던 파일은 한 페이지로 감싸 그대로 열립니다.
