# 5E 기능 레퍼런스 (AI 도우미용)

이 문서는 AI 도우미 챗봇이 사용법 질문에 답할 때 근거로 사용하는 기능 요약이다.
여기에 없는 내용은 챗봇이 답하지 않는다.

## 프로그램 개요

5E는 과학/물리 교사가 시험지·학습지에 넣을 그림을 빠르게 그리는 웹 기반 다이어그램
드로잉 도구다. 브라우저에서 실행되며 설치가 필요 없다.

## 화면 구성

- 왼쪽 서랍: 공통 도구, 기호 라이브러리(카테고리별), 고급 기능 버튼
- 중앙: 캔버스. 상단 툴바에 되돌리기/다시 실행, 파일 메뉴, 설정 메뉴, 전체화면,
  흑백 모드 토글, 현재 zoom 표시가 있다
- 오른쪽 서랍: 인스펙터(선택한 객체의 속성 편집), 레이어 패널(눈 아이콘으로 표시/숨김)
- 왼쪽·오른쪽 서랍은 툴바의 서랍 토글 버튼으로 접고 펼 수 있다

## 공통 도구와 단축키

- 선택 (V) / 사각형 (S) / 타원 (O) / 직각삼각형 (Y) / 직선 (L) / 꺾은선 (P)
- 곡선 (C) / 텍스트 (T) / 자유그리기 (F) / 회전 (R)
- 자르기 도구: 가위·칼·올가미 방식으로 객체를 잘라낸다
- 되돌리기 Ctrl+Z, 다시 실행 Ctrl+Y 또는 Ctrl+Shift+Z
- 오브젝트 검색 Ctrl+F: 도구·기호를 이름으로 검색해 바로 사용
- 그룹 해제 Shift+G (선택 도구 V 상태에서, 같은 그룹의 객체를 모두 선택한 뒤)

## 기호 라이브러리 (템플릿)

- 왼쪽 서랍에 카테고리별로 접히는 섹션이 있다: 공통, 함수, 회로, 광학, 역학, 전자기학
- 버튼을 클릭하면 해당 기호(예: 전지, 전구, 렌즈, 도르래, 도선 등)가 캔버스에 삽입된다
- 라벨러(지시선 딸린 라벨), 각도 호, 직각 표시, 길이 표시, 화살표 선도 지원한다

## 함수 그래프와 좌표평면

- 기호 라이브러리의 "함수" 카테고리에서 함수그래프와 좌표평면을 삽입한다
- 함수그래프를 선택하면 함수 입력 모달이 열리고 y=x^2 같은 식을 입력한다
- 한 그래프에 여러 함수를 함께 넣을 수 있다 (다중 함수 입력)
- LaTeX 수식 팔레트와 도움말이 모달에 내장되어 있다
- 좌표평면은 상세 설정 모달에서 원점 라벨 등 세부를 조정할 수 있다

## 텍스트와 수식

- 텍스트 도구(T)로 캔버스에 텍스트를 넣는다
- LaTeX 문법 수식을 지원한다. \sin 같은 함수명은 정자체로, 숫자도 정자체로 렌더된다

## 파일 메뉴

- 프로젝트 저장: 현재 작업을 나중에 다시 편집할 수 있는 프로젝트 파일(JSON)로 저장
- 프로젝트 불러오기: 저장한 프로젝트 파일을 열어 이어서 편집
- 이미지 불러오기: JPG, PNG 등 이미지 파일을 캔버스에 불러와 배치 (일반 이미지 객체)
- 이미지로 내보내기: 현재 그림을 PNG 또는 JPG 파일로 저장
- 참고: 프로젝트 파일에는 객체·가이드·레이어·아트보드가 저장되고,
  되돌리기 기록과 선택 상태는 저장되지 않는다

## 내보내기 상세

- PNG는 200/300/400 dpi 중 선택할 수 있어 시험지 인쇄 품질에 맞출 수 있다
- SVG 내보내기 지원
- 영역 지정 내보내기: 캔버스 일부만 골라 내보낼 수 있다
- 파일명에는 날짜시각(YYYYMMDD_HHmm)이 자동으로 붙는다

## 이미지 관련 기능

- 이미지 불러오기(파일 메뉴) 또는 Ctrl+V(클립보드 붙여넣기)로 이미지를 넣으면
  일반 이미지 객체가 된다. 이 상태에서는 이미지 속 선을 개별 편집할 수 없다
- 고급 기능 → 이미지 객체화: PNG 그림을 분리된 편집 가능한 선 객체들로 변환한다.
  흰 배경에 선이 분명한 그림(선화)일수록 결과가 좋고, 사진은 부정확하다.
  글자로 판정된 덩어리는 남기기/지우기/텍스트 객체로 대체 중에서 고를 수 있다.
  변환 결과는 하나의 그룹으로 삽입되며 Shift+G로 그룹을 해제할 수 있다.
  삽입은 되돌리기 한 번(Ctrl+Z)으로 취소된다
- 이미지 오려내기: 이미지 객체의 편집 모드에서 사각형/자유 영역을 지울 수 있다

## 설정

- 설정 메뉴 → 기본값 설정: 새 객체의 기본 스타일 등을 저장(localStorage에 보존)
- 흑백 모드 토글: 툴바 오른쪽 버튼
- 다크/라이트 테마 지원

## 스냅과 보조 도구

- 객체를 그리거나 옮길 때 스냅이 동작해 다른 객체·격자에 정렬된다
- 가이드와 눈금자(룰러)를 지원한다

## 제한 사항

- 내보낸 파일의 글꼴 모양은 보는 컴퓨터에 설치된 폰트에 따라 달라질 수 있다
- 이미지 객체화는 흑백 선화 전용에 가깝고, 사진·복잡한 이미지는 부정확하다
- AI 도우미는 무료 외부 AI를 사용하므로 질문 사이에 몇 초 간격이 필요하다

## AI 도우미 자체에 대한 안내

- 질문 내용은 프록시(중계 서버)를 거쳐 외부 무료 AI 서비스(Pollinations.AI)로 전송된다.
  학생 개인정보나 출제 전 문항 내용은 입력하지 않는 것이 좋다
- 처음 쓸 때 하단 [프록시 설정]에 배포한 Worker 주소를 한 번 입력해야 한다
- AI 도우미는 이 레퍼런스에 있는 5E 사용법 질문에만 답한다
