## 1. 시작하기 — 화면 구성·선택·이동·실행취소·단축키 총정리

5E는 과학 시험 문항과 수업 자료의 도해(그림)를 만드는 웹 기반 편집기입니다. 별도 설치 없이 브라우저에서 열리며, 왼쪽에서 도구를 고르고 → 가운데 캔버스에 그리고 → 오른쪽에서 속성을 다듬는 3단 구성으로 되어 있습니다. 이 장은 v0.54.14 기준으로 화면의 각 부분, 오브젝트 선택·이동, 실행 취소, 그리고 전체 단축키를 정리합니다.

> **용어 안내**: 이 안내서에서 "오브젝트"는 캔버스에 그린 도형·선·텍스트 등 하나하나를 말합니다. "아트보드"는 실제로 내보내지는 종이 영역(흰 판)을 말합니다. 캔버스의 좌표 단위는 mm이며(1칸 = 1mm), 아트보드 기본 크기는 90×60mm입니다(최대 200×200mm).

### 1.1 화면 구성

**무엇인가**: 5E의 화면은 상단 툴바 · 좌측 도구 패널 · 중앙 캔버스 · 우측 속성 패널 · 하단 바 · 푸터의 6개 구역으로 이루어져 있습니다.

**처음 열었을 때 확인 순서**

1. **상단 툴바** — 왼쪽부터 5E 로고, 되돌리기(↶)/다시 실행(↷) 버튼, **파일 ▾** 메뉴(프로젝트 저장·불러오기, 이미지 불러오기, 이미지로 내보내기), **설정 ▾** 메뉴(기본값 설정, 설정 저장·불러오기), 현재 배율 표시(`zoom 1.00×`), 전체화면 버튼, 흑백(어두운) 모드 전환 버튼이 있습니다.
2. **좌측 도구 패널** — 맨 위 **과목 선택**(물리학/화학/생명과학/지구과학)을 바꾸면 강조색 테마와 아래 "과목별 오브젝트" 목록이 함께 바뀝니다. 그 아래로 **공통 도구**(선택·회전·자르기·도형·선·텍스트 등 버튼 격자), **과목별 오브젝트**, **퍼스널 오브젝트**(내가 저장한 오브젝트 묶음), **고급 기능**(이미지 객체화, 오브젝트 저장, 전체 통일/수정) 섹션이 있습니다. 각 섹션 제목을 클릭하면 접고 펼 수 있습니다.
3. **중앙 캔버스** — 위와 왼쪽에 mm 눈금자가 항상 표시되고, 가운데 흰 아트보드가 놓입니다. 마우스를 올리면 눈금자에 십자선 위치가 표시됩니다.
4. **하단 바** — **오브젝트 검색**, **기출문항 검색** 버튼, **격자** 토글(켜면 진하기 1~10, 간격 5~50mm 조절 칸이 나타남), **⌖**(아트보드 중앙 고정) 버튼이 있고, 오른쪽 끝에는 현재 도구의 한 줄 조작 안내(예: "선택 · 클릭=선택 · 드래그=이동 · Shift 드래그=스냅")가 뜹니다.
5. **우측 속성 패널(인스펙터)** — 아무것도 선택하지 않으면 "선택된 오브젝트 없음"이 보이고, 오브젝트를 선택하면 그 종류에 맞는 속성 섹션이 나타납니다. 패널의 왼쪽 경계를 드래그하면 폭을 조절할 수 있습니다.
6. **푸터** — 앱 이름·버전·날짜가 항상 표시됩니다.

**화면 구성 관련 단축키**

| 동작 | 키 |
|---|---|
| 전체화면 켜기/끄기 | `Alt+Enter` |
| 흑백(어두운) 모드 전환 | 툴바 반달 모양 버튼 (단축키 없음) |
| 좌/우 패널 열기(모바일·좁은 화면) | 툴바의 `☰ 도구` / `속성` 버튼, `Esc`로 닫기 |

**팁·주의**

- 격자와 눈금자는 화면 참고용일 뿐, 내보낸 이미지에는 포함되지 않습니다.
- 좁은 화면(폭 768px 미만)에서는 좌우 패널이 서랍(드로어)으로 바뀌어 캔버스 위에 겹쳐 열립니다. 한 번에 하나만 열립니다.
- 흑백 모드 설정과 개인 기본값은 브라우저에 저장되므로 다음 접속 때도 유지됩니다.

#### 안내선(가이드) 만들기

**무엇인가**: 눈금자를 클릭해 만드는 편집용 보조선입니다. 내보내기에는 포함되지 않습니다.

1. 위쪽(가로 위치용) 또는 왼쪽(세로 위치용) **눈금자를 클릭**하면 그 위치에 안내선이 생깁니다.
2. 눈금자 근처에서 안내선을 **드래그**하면 위치를 옮길 수 있습니다. 캔버스 위에서도 안내선 위에 포인터를 올리면 ↕/↔ 커서가 나타나 잡아 옮길 수 있습니다.
3. 눈금자 위의 **좌표 배지를 클릭**하면 숫자 입력 칸이 열립니다. mm 값을 직접 입력하고 `Enter`로 확정, `Esc`로 취소합니다.
4. 안내선이 선택된 상태에서 `Delete`를 누르면 삭제됩니다.

| 동작 | 방법 |
|---|---|
| 안내선 생성 | 눈금자 클릭 |
| 안내선 이동 | 안내선 드래그 |
| 좌표 직접 입력 | 좌표 배지 클릭 → 숫자 입력 → `Enter` |
| 안내선 삭제 | 안내선 선택 후 `Delete` |

- **주의**: 안내선 생성·삭제는 실행 취소(`Ctrl+Z`) 대상이 아닙니다.

### 1.2 오브젝트 선택

**무엇인가**: 선택 도구(V)로 캔버스의 오브젝트를 골라 이동·편집 대상으로 삼는 기본 동작입니다.

**사용 절차**

1. 좌측 패널의 화살표 버튼을 누르거나 `V` 키를 눌러 **선택 도구**를 켭니다. (도형을 하나 그리고 나면 자동으로 선택 도구로 돌아옵니다.)
2. 오브젝트를 **클릭**하면 선택됩니다. 여러 오브젝트가 겹쳐 있으면 맨 위 것이 선택됩니다.
3. **Shift+클릭**으로 선택에 추가하거나 뺄 수 있습니다(다중 선택).
4. **빈 곳에서 드래그**하면 파란 점선 사각형(범위 선택)이 그려지고, 놓는 순간 그 범위에 걸린 오브젝트들이 모두 선택됩니다.
5. **빈 곳을 클릭**하면 선택이 해제됩니다.

**그룹과 더블클릭**

- 그룹으로 묶인 오브젝트를 클릭하면 **그룹 전체**가 선택됩니다(초록 테두리).
- 그룹 안의 한 멤버만 만지고 싶으면 그 멤버를 **더블클릭**하세요. 주황색 "지목" 상태가 되며, 이 상태에서는 이동·변형이 잠기고 그룹 해제(`Shift+G`)나 인스펙터 편집만 가능합니다.
- **텍스트/수식** 오브젝트 더블클릭 → 내용 편집, **라벨러** 더블클릭 → 라벨 편집, **각도호** 더블클릭 → 각도 라벨 편집, **좌표평면** 더블클릭 → 상세 설정 창.

**선택 관련 조작 정리**

| 동작 | 방법 |
|---|---|
| 선택 도구 켜기 | `V` |
| 하나 선택 | 클릭 |
| 선택에 추가/제거 | `Shift`+클릭 |
| 범위 선택(마퀴) | 빈 곳에서 드래그 |
| 선택 해제 | 빈 곳 클릭 |
| 그룹 멤버 지목 | 그룹 멤버 더블클릭 |
| 그룹 묶기 / 풀기 | `G` / `Shift+G` (선택 도구에서) |
| 잠금/해제 토글 | `K` (선택 도구에서) |
| 오브젝트 이름으로 찾기 | `Ctrl+F` (오브젝트 검색 창) |

**팁·주의**

- 선택은 **현재 활성 레이어**의 오브젝트에만 적용됩니다. 레이어는 3개(레이어 1~3)이며 우측 인스펙터의 레이어 섹션에서 전환합니다. "분명 보이는데 선택이 안 될 때"는 먼저 그 오브젝트가 다른 레이어에 있지 않은지 확인하세요.
- 잠긴(`K`) 오브젝트는 삭제·변형되지 않습니다. 잠금을 풀 때도 선택 후 `K`입니다.
- `Esc`는 선택 해제가 아니라 "진행 중인 그리기 취소·창 닫기"에 씁니다. 선택 해제는 빈 곳 클릭입니다.

### 1.3 오브젝트 이동·기본 변형

**무엇인가**: 선택한 오브젝트를 드래그나 화살표 키로 옮기고, 핸들로 크기를 바꾸고, 회전 도구로 돌리는 동작입니다.

**드래그로 이동**

1. 선택 도구(`V`) 상태에서 오브젝트를 클릭해 선택합니다.
2. 선택된 오브젝트 위에서 포인터가 손 모양(grab)으로 바뀌면 **누른 채 끌어** 원하는 위치에 놓습니다. 다중 선택·그룹은 함께 움직입니다.
3. 끄는 동안 **Shift를 누르면** 주변 오브젝트의 끝점·모서리 등에 달라붙는 **스냅**이 작동하고 빨간 점으로 스냅 지점이 표시됩니다.

**화살표 키로 미세 이동**

1. 오브젝트를 선택합니다.
2. 화살표 키를 누르면 **0.5mm**씩, `Ctrl`+화살표는 **5mm**씩 움직입니다. 꾹 누르고 있으면 연속 이동하지만 실행 취소 기록은 한 번만 남습니다.

**크기 조절(핸들)**

1. 오브젝트를 **하나만** 선택하면 테두리에 핸들(작은 사각 점)이 나타납니다.
2. 핸들을 드래그해 크기를 바꿉니다. 선 종류는 끝점 핸들을 끌며, 이때 `Shift`를 누르면 다른 선의 끝점이나 도형 가장자리에 스냅됩니다.
3. 그룹 전체를 선택한 상태에서는 결합 테두리의 핸들로 **비율이 고정된 일괄 크기 조절**이 됩니다.

**회전(회전 도구)**

1. `R` 키(또는 회전 버튼)로 **회전 도구**를 켭니다.
2. 오브젝트를 선택하고 **모서리(꼭짓점) 핸들**을 드래그해 돌립니다. `Ctrl`을 누르면 15° 단위로 끊어 돌아갑니다.
3. 회전 도구 상태에서 `PageUp`/`PageDown`을 누르면 +5°/−5°씩 정밀 회전, **화살표 키**를 누르면 좌우(←/→) 또는 상하(↑/↓) **반전**이 됩니다.

**이동·변형 단축키/모디파이어 표**

| 동작 | 키·조작 |
|---|---|
| 이동 | 선택 후 드래그 |
| 이동 중 스냅 | 드래그 중 `Shift` |
| 미세 이동 0.5mm / 5mm | 화살표 / `Ctrl`+화살표 |
| 선 굵기 ±0.1mm | `Shift+↑` / `Shift+↓` |
| 회전 도구 | `R` |
| 회전 15° 스냅 | 회전 드래그 중 `Ctrl` |
| 5°씩 회전 | (회전 도구) `PageUp` / `PageDown` |
| 좌우/상하 반전 | (회전 도구) `←→` / `↑↓` |
| 삼각형 상하 반전 | 삼각형 선택 후 `F` |
| 앞으로/뒤로 (겹침 순서) | (선택 도구) `PageUp` / `PageDown` |
| 삭제 | `Delete` |
| 복사 / 붙여넣기 | `Ctrl+C` / `Ctrl+V` (마우스 위치에 붙음) |
| 스타일·각도만 복사 / 적용 | `Shift+C` / `Shift+V` |

**팁·주의**

- `Ctrl+V` 붙여넣기는 **마우스 포인터가 있는 자리**를 중심으로 놓입니다. 원하는 위치에 포인터를 두고 붙여넣으세요.
- `Shift+C`/`Shift+V`는 내용·위치는 그대로 두고 **굵기·색·글꼴·각도 같은 겉모습만** 다른 오브젝트로 옮기는 기능입니다. 여러 개를 선택해 한꺼번에 적용할 수 있습니다.
- 인스펙터에서 "위치 잠금"된 오브젝트는 드래그·화살표로 움직이지 않습니다(크기 조절 시에도 중심이 고정됩니다).

---

### 1.4 화면 이동과 확대·축소

**무엇인가**: 오브젝트는 그대로 두고 보는 창(뷰포트)만 움직이는 기능입니다. 현재 배율은 툴바의 `zoom 1.00×` 표시로 확인합니다.

**사용 절차**

1. **마우스 휠**을 굴리면 화면이 위아래로 이동합니다. `Shift`+휠은 좌우 이동입니다.
2. **`Ctrl`+휠**로 확대·축소합니다. 마우스 포인터 위치를 중심으로 배율이 바뀝니다.
3. 자유롭게 끌어 이동하려면 **Space를 누른 채 드래그**하거나 **휠(가운데) 버튼 드래그**를 사용합니다.
4. 아트보드를 화면 정중앙에 고정하려면 하단 바의 **⌖ 버튼**을 누릅니다. 다시 누르면 해제됩니다.

**화면 조작 표**

| 동작 | 키·조작 |
|---|---|
| 상하 이동 | 휠 |
| 좌우 이동 | `Shift`+휠 |
| 확대/축소 | `Ctrl`+휠 (포인터 중심) |
| 자유 이동(팬) | `Space`+드래그 또는 휠 버튼 드래그 |
| 아트보드 중앙 고정 | 하단 바 `⌖` 버튼 (토글) |
| 전체화면 | `Alt+Enter` |

**팁·주의**

- 확대는 최대 **100배**, 축소는 아트보드가 화면의 절반보다 작아지지 않는 선까지만 됩니다. 화면 밖으로 아트보드가 완전히 사라지지 않도록 이동 범위도 자동 제한됩니다.
- **⌖(중앙 고정)이 켜져 있는 동안에는 화면 이동이 잠깁니다.** 휠·드래그로 화면이 안 움직인다면 이 버튼이 켜져 있지 않은지 확인하세요. 이때 확대·축소는 아트보드 중심 기준으로 동작합니다.
- 화면 이동·확대는 그림 데이터를 바꾸지 않으므로 실행 취소 대상이 아닙니다.

---

### 1.5 실행 취소 / 다시 실행

**무엇인가**: 오브젝트에 가한 변경(생성·이동·크기·회전·삭제·속성 변경 등)을 한 단계씩 되돌리거나 다시 적용합니다.

**사용 절차**

1. 방금 한 작업을 되돌리려면 `Ctrl+Z`를 누르거나 툴바의 **↶** 버튼을 클릭합니다.
2. 되돌린 것을 다시 적용하려면 `Ctrl+Shift+Z` 또는 **↷** 버튼을 사용합니다.
3. 되돌릴 것이 없으면 버튼이 회색(비활성)으로 표시됩니다.

**단축키 표**

| 동작 | 키 |
|---|---|
| 실행 취소 | `Ctrl+Z` (또는 툴바 ↶) |
| 다시 실행 | `Ctrl+Shift+Z` (또는 툴바 ↷) |

**팁·주의**

- 다시 실행 단축키는 `Ctrl+Shift+Z` **하나뿐**입니다. `Ctrl+Y`는 지원하지 않습니다.
- 실행 취소를 한 뒤 **새로운 편집을 하면 "다시 실행" 기록은 지워집니다**(일반적인 편집기와 동일).
- 기록되는 것은 오브젝트 변경뿐입니다. **선택 상태, 화면 이동·확대, 격자 켜기, 안내선**은 기록되지 않습니다.
- 화살표 키를 꾹 눌러 연속 이동한 경우, 한 번의 `Ctrl+Z`로 누르기 시작 전 위치까지 한꺼번에 돌아갑니다. 드래그 이동·회전도 드래그 한 번이 한 단계입니다.

## 2. 그리기 도구 — 선·도형·자유곡선·스냅·자·각도기·가위(절단)

이 장에서는 5E의 왼쪽 도구 패널 맨 위 **「공통 도구」** 묶음에 있는 기본 그리기 도구를 다룹니다. 과목(물리학·화학·생명과학·지구과학)과 상관없이 항상 같은 방식으로 동작합니다.

### 2.0 모든 그리기 도구에 적용되는 공통 규칙

본격적인 설명 전에, 어떤 도구를 쓰든 똑같이 적용되는 규칙 여섯 가지를 먼저 알아 두세요.

1. **도구는 왼쪽 패널 버튼을 클릭하거나 단축키를 눌러 켭니다.** 켜진 도구의 버튼은 파랗게 강조됩니다.
2. **하나를 그리고 나면 자동으로 선택(V) 도구로 돌아옵니다.** 같은 도구로 연달아 그리려면 단축키를 다시 누르면 됩니다.
3. **캔버스 아래쪽 안내 바**에 현재 도구의 조작법이 한 줄로 항상 표시됩니다. 조작법이 헷갈리면 이곳을 보세요.
4. **Space 키를 누른 채 드래그하면 화면 이동(팬)**이 되고, 그동안은 아무것도 그려지지 않습니다.
5. **너무 작게 그린 것(약 0.3mm 미만)은 무시됩니다.** 실수로 캔버스를 살짝 클릭해도 의미 없는 점이 생기지 않습니다.
6. **되돌리기는 Ctrl+Z 한 번이면 충분합니다.** 방금 그린 객체 하나가 통째로 사라집니다. (Shift+Z = 다시 실행)

캔버스의 좌표 단위는 **mm**입니다. 화면에서 10mm로 그린 선은 인쇄물에서도 10mm입니다.

| 공통 조작 | 동작 |
|---|---|
| `V` | 선택 도구로 복귀 |
| `R` | 회전 도구 (선택한 객체를 드래그해 회전) |
| `Space` + 드래그 | 화면 이동(그리기 중단 없음) |
| `Esc` | 진행 중인 그리기 취소 |
| `Ctrl+Z` / `Shift+Z` | 되돌리기 / 다시 실행 |

---

### 2.1 직선 (L)

두 점을 클릭해 곧은 선 하나를 그리는 도구입니다.

**사용 절차**

1. 왼쪽 패널의 「직선」 버튼을 클릭하거나 `L`을 누릅니다.
2. 선이 시작될 곳을 **클릭**합니다. (누른 채 끌지 않아도 됩니다.)
3. 마우스를 움직이면 시작점에서 커서까지 미리보기 선이 따라옵니다.
4. 선이 끝날 곳을 **클릭**하면 완성되고, 선택 도구로 돌아옵니다.

**단축키·보조키**

| 키 | 시점 | 동작 |
|---|---|---|
| `L` | — | 직선 도구 켜기 |
| `Shift` | 끝점 클릭 전 누른 채 | **끝점 스냅**: 끝점이 근처 도형의 꼭짓점·변·곡면 위에 정확히 달라붙음 (빨간 점으로 붙을 위치 미리 표시) |
| `Ctrl` | 끝점 클릭 전 누른 채 | **각도 스냅**: 시작점 기준 15° 단위로만 기울기 고정 (0°/90°는 오차 없이 정확한 수평·수직) |
| `Esc` | 첫 점만 찍은 상태 | 취소 |

**팁·주의**

- `Shift`와 `Ctrl`을 함께 누르면 **Shift(끝점 스냅)가 우선**합니다.
- 광선(빛) 그림을 그릴 때 `Shift`를 누르면 렌즈 면이나 「물체」 화살표의 **화살촉 끝**에 광선이 정확히 닿게 붙습니다.
- 그린 뒤 화살촉(끝/시작/양쪽), 점선, 굵기 등은 오른쪽 인스펙터에서 바꿉니다.

---

### 2.2 꺾은선 (P)

여러 점을 차례로 클릭해 꺾인 선(빗면·계단·그래프 개형 등)을 그리는 도구입니다.

**사용 절차**

1. 「꺾은선」 버튼을 클릭하거나 `P`를 누릅니다.
2. 꼭짓점이 될 자리를 하나씩 **클릭**해 점을 추가합니다. 마지막 점에서 커서까지 미리보기 선이 따라옵니다.
3. 다 찍었으면 **더블클릭** 또는 `Enter`로 완성합니다. (점이 2개 이상이어야 합니다.)

**단축키·보조키**

| 키 | 동작 |
|---|---|
| `P` | 꺾은선 도구 켜기 |
| 클릭 | 꼭짓점 추가 |
| 더블클릭 / `Enter` | 완성 |
| `Esc` | 전체 취소 (아무것도 남지 않음) |
| `Ctrl` + 클릭 | 직전 꼭짓점 기준 15° 각도 스냅 |

**팁·주의**

- 더블클릭으로 끝내도 점이 이중으로 찍히지 않습니다(자동 정리).
- 완성 후 인스펙터에서 **「닫기」**를 켜면 처음과 끝이 이어진 채워진 다각형이 되고, 채움 색·무늬(점·빗금 등)를 지정할 수 있습니다.
- 인스펙터의 **경사면처리(모서리 둥글리기)** 옵션을 켜면 꺾이는 모서리가 부드러운 곡면으로 바뀌어 빗면·언덕 그림에 좋습니다.

---

### 2.3 곡선 (C)

여러 점을 클릭하면 그 점들을 부드럽게 지나는 곡선을 만들어 주는 도구입니다.

**사용 절차**

1. 「곡선」 버튼을 클릭하거나 `C`를 누릅니다.
2. 곡선이 지나갈 자리를 하나씩 **클릭**합니다. 미리보기가 처음부터 곡선 모양으로 표시되어 결과와 똑같습니다.
3. **더블클릭** 또는 `Enter`로 완성합니다.

**단축키·보조키**

| 키 | 동작 |
|---|---|
| `C` | 곡선 도구 켜기 |
| 클릭 | 지나갈 점(앵커) 추가 |
| 더블클릭 / `Enter` | 완성 |
| `Esc` | 전체 취소 |

**팁·주의**

- 점을 **적게, 듬성듬성** 찍을수록 매끈한 곡선이 나옵니다. 파동·궤적 그림은 마루/골 지점만 찍는 것이 요령입니다.
- 곡선에는 `Ctrl` 각도 스냅이 없습니다(곧게 꺾을 부분이 있으면 꺾은선을 쓰세요).
- 인스펙터에서 「닫기」를 켜면 매끄럽게 닫힌 채움 도형이 됩니다.

---

### 2.4 도형 — 사각형(S)·타원(O)·직각삼각형(Y)

대각선 방향으로 드래그해 크기가 있는 도형을 그리는 도구 세 가지입니다. 세 도구 모두 조작법이 동일합니다.

**사용 절차**

1. 「사각형」(`S`), 「타원」(`O`), 「직각삼각형」(`Y`) 중 하나를 켭니다.
2. 도형의 한 모서리가 될 지점에서 **마우스를 누릅니다**.
3. 반대쪽 모서리까지 **끈 뒤 놓으면** 완성됩니다. 어느 방향으로 끌어도 됩니다.

**단축키·보조키**

| 키 | 동작 |
|---|---|
| `S` | 사각형 (주의: `R`이 아니라 `S`입니다. `R`은 회전 도구) |
| `O` | 타원 |
| `Y` | 직각삼각형 |
| `Shift` + 드래그 | **정비율 고정**: 정사각형 / 정원 (사각형·타원만) |

**팁·주의**

- 새 도형은 기본적으로 **흰색 불투명 채움 + 검정 테두리**로 만들어집니다. 채움을 없애거나 회색조·무늬로 바꾸는 것은 인스펙터에서 합니다. (기본 테두리 굵기 등은 설정 메뉴의 「기본값 설정」에서 바꿀 수 있습니다.)
- **직각삼각형의 직각 위치는 드래그 방향으로 정해집니다.** 오른쪽으로 끌면 직각이 왼쪽 아래, 왼쪽으로 끌면 오른쪽 아래에 옵니다. 그린 뒤에는 삼각형을 선택하고 `F`를 눌러 상하 반전할 수 있습니다.
- 빗면 문제의 경사면은 「직각삼각형 + 회전」 또는 「꺾은선 + 경사면처리」 두 방법이 모두 가능합니다.

---

### 2.5 자유 그리기 (D)

마우스로 자유롭게 그린 궤적을 매끈한 **닫힌 곡선 도형**으로 만들어 주는 도구입니다.

**사용 절차**

1. 「자유 그리기」 버튼을 클릭하거나 `D`를 누릅니다.
2. 원하는 모양을 따라 **누른 채 드래그**합니다. 드래그 중에는 가는 선으로 궤적이 보입니다.
3. 놓으면 궤적이 자동으로 다듬어지고(잔떨림 제거), 처음과 끝이 이어진 닫힌 도형으로 확정됩니다.

**단축키·보조키**

| 키 | 동작 |
|---|---|
| `D` | 자유 그리기 도구 켜기 |

**팁·주의**

- 완성된 도형은 기본이 **테두리 없는 흰색 채움**입니다. 주된 용도가 *불러온 이미지의 일부를 가리는 것*이기 때문입니다(기출 그림에서 라벨을 지우고 새로 붙일 때 유용). 윤곽선이 필요하면 인스펙터에서 테두리를 켜세요.
- 너무 짧게 긋거나 점만 찍으면(다듬은 뒤 점 3개 미만) 아무것도 만들어지지 않습니다.
- 손떨림은 자동으로 걸러지므로 천천히, 크게 긋는 편이 결과가 좋습니다.

---

### 2.6 스냅 — 오차 없이 정확히 붙이기

스냅은 별도의 도구가 아니라, **Shift 키를 누르고 있는 동안만 켜지는 자석 기능**입니다. 세 가지 상황에서 동작하며, 붙게 될 위치는 항상 **빨간 점(선)** 으로 미리 보여줍니다.

**① 객체를 옮길 때 (선택 도구 V + Shift 드래그)**

1. 객체를 선택하고 드래그를 시작합니다.
2. `Shift`를 누른 채 다른 객체 가까이 가져가면 빨간 표시가 나타나고, 충분히 가까워지면 자석처럼 달라붙습니다.
3. 붙는 방식은 객체 종류에 따라 자동으로 정해집니다.
   - 선·꺾은선·곡선: **끝점끼리** 정확히 연결
   - 사각형·삼각형: 상대 도형의 **변에 면이 닿도록** 자동 회전하며 안착 (빗면 위에 물체 상자를 얹을 때)
   - 원·타원: 변이나 선에 **접하도록** 안착 / 직선을 원에 가져가면 **접선**으로 안착
   - 광학 「물체」 화살표: **화살촉 끝**이 렌즈·거울·선 표면에 정확히 닿음
   - 점(node): 점의 중심이 가장 가까운 변·곡면·꼭짓점 위로

**② 직선을 그릴 때 (2.1 참조)** — `Shift` = 끝점을 다른 도형에 스냅, `Ctrl` = 15° 각도 스냅.

**③ 이미 그린 선의 끝점을 수정할 때** — 선을 선택하고 끝점 핸들을 `Shift`를 누른 채 드래그하면 ①과 같은 끝점 스냅이 동작합니다. 추가로, 끝점을 원·사각형의 **중심을 향해** 겨누면 선이 중심을 정확히 지나도록 경계면 위에 스냅됩니다(원의 반지름 방향 광선·힘 화살표에 유용).

| 키 | 동작 |
|---|---|
| `Shift` (드래그/그리기 중) | 스냅 켜기 — 누르고 있는 동안만 |
| `Ctrl` (선 계열 그리기 중) | 15° 각도 스냅 |

**팁·주의**

- 스냅 대상은 **화면에 보이는 모든 레이어**의 객체입니다. 잠근 객체도 스냅 기준으로 쓸 수 있습니다(고정 기준선으로 활용).
- 빨간 표시가 보이지만 아직 붙지 않은 상태(거리가 애매할 때)도 있습니다. 더 가까이 가져가면 달라붙습니다.
- 스냅을 원치 않으면 그냥 `Shift`를 떼면 됩니다.

### 2.7 자 (측정 가이드)

캔버스 위에 올려놓는 **눈금자 모양의 측정 가이드 객체**입니다. 화면 확대/축소와 무관하게 실제 mm 눈금을 보여줍니다.

**사용 절차**

1. 왼쪽 패널의 「자」 버튼을 클릭합니다. (키보드 단축키는 없습니다.)
2. 캔버스에서 가로로 **드래그**합니다. 드래그한 가로 길이가 자의 길이가 되고, 세로 폭은 10mm 띠로 고정됩니다.
3. 놓으면 완성됩니다. 눈금이 새겨진 **측정 가장자리는 위쪽**입니다.

**설정(인스펙터 「자·각도기」 섹션)**

| 항목 | 내용 |
|---|---|
| 눈금 간격(자) | 눈금 하나 사이의 간격(mm). 기본 10mm — **1로 바꾸면 실제 자처럼 1mm 간격**이 되고 5mm·1cm 눈금이 길게 구분됩니다 |
| 투명도 | 0~100%. 반투명하게 해 두면 아래 그림을 가리지 않습니다 |

**팁·주의**

- 새 자의 기본 눈금 간격은 설정 메뉴 → **「기본값 설정」**에서 미리 바꿔 둘 수 있습니다.
- 회전 도구(`R`)로 기울여 빗면 길이를 재거나, 대각선 거리를 확인할 수 있습니다.
- 띠가 충분히 높으면 1cm마다 숫자가 함께 표시됩니다.

---

### 2.8 각도기

캔버스 위에 올려놓는 **반원 각도기 모양의 측정 가이드 객체**입니다.

**사용 절차**

1. 왼쪽 패널의 「각도기」 버튼을 클릭합니다. (키보드 단축키는 없습니다.)
2. 캔버스에서 **드래그**합니다. 드래그한 크기(가로·세로 중 큰 쪽)가 반지름이 되고, 위로 볼록한 반원이 만들어집니다. 아래 밑변이 0°–180° 지름선입니다.
3. 놓으면 완성됩니다.

**설정(인스펙터 「자·각도기」 섹션)**

| 항목 | 내용 |
|---|---|
| 눈금 간격(각도기) | 방사형 눈금 간격(°). 기본 10° — 30°마다 각도 숫자가 표시됩니다 |
| 투명도 | 0~100% |

**팁·주의**

- 각도기의 중심(밑변 가운데)을 재고 싶은 꼭짓점에 맞춘 뒤, 회전 도구(`R`)로 밑변을 기준선에 맞추면 임의 방향의 각도를 잴 수 있습니다.
- 가로:세로 비율이 잠겨 있어 크기를 조절해도 반원이 찌그러지지 않습니다.
- 각도를 **재는** 것이 아니라 그림에 각도를 **표시**하려면 아래 2.10의 「각도 호」를 쓰세요.

---

### 2.9 가위 — 자르기 (E)

캔버스의 객체 위로 선을 그어, 지나간 자리에서 객체를 **두 조각 이상으로 잘라내는** 도구입니다.

**사용 절차**

1. 「자르기」 버튼을 클릭하거나 `E`를 누릅니다. 커서가 십자선으로 바뀝니다.
2. 도구가 켜지면 모든 객체에 점선 상자가 표시됩니다. **파란 상자 = 자를 수 있음, 회색 상자 = 자를 수 없음**(텍스트·이미지 등).
3. 자르고 싶은 객체를 **가로지르도록 드래그**합니다. 그은 경로는 빨간 점선으로, 실제로 잘리게 될 지점(교차점)은 **빨간 점**으로 실시간 표시됩니다.
4. 놓는 순간 경로가 지나간 모든 대상이 교차점에서 잘리고, **잘린 조각들이 자동으로 선택**됩니다. 바로 끌어서 벌리거나 `Delete`로 일부를 지울 수 있습니다.

**단축키·보조키**

| 키 | 동작 |
|---|---|
| `E` | 자르기 도구 켜기 |
| 드래그 | 자유곡선으로 자르기 |
| `Shift` + 드래그 | **직선**으로 자르기 |
| `Shift`+`Ctrl` + 드래그 | 직선 + 15° 각도 스냅으로 자르기 |

**자를 수 있는 것과 잘리는 방식**

| 대상 | 결과 |
|---|---|
| 직선·꺾은선·열린 곡선 | 교차점에서 나뉜 **열린 조각들** |
| 사각형·타원·삼각형·닫힌 꺾은선/곡선 (채움 있음) | 잘린 단면이 닫힌 **채워진 두 조각** (케이크 자르듯) |
| 채움 없는 닫힌 도형 | 윤곽선만 열린 조각으로 분리 |
| 텍스트·수식·이미지·회로 기호 등 | 잘리지 않음 (회색 상자) |

**팁·주의**

- 경로가 지나가는 **모든** 자를 수 있는 객체가 한꺼번에 잘립니다. 원치 않는 객체 위를 지나지 않게 긋고, 실수했다면 `Ctrl+Z` 한 번으로 전부 복구됩니다.
- 빨간 점이 하나도 보이지 않으면 놓아도 아무 일도 일어나지 않습니다(안전).
- 원을 반원 두 개로 나누거나, 그래프 곡선의 일부 구간만 남기고 지우는 용도로 특히 유용합니다.

### 2.10 함께 알아두면 좋은 각도 표시 도구 — 각도 호(A)·직각 표시(Shift+A)

각도기(2.8)가 각도를 *재는* 가이드라면, 이 두 도구는 완성 그림에 들어갈 **각도 기호를 그리는** 도구입니다.

**각도 호 (A)** — 꼭짓점에 θ가 붙은 부채꼴 호를 그립니다.

1. `A`를 누르고, ① 각의 **꼭짓점**을 클릭 → ② 호가 시작될 **한 변 위의 점**을 클릭(꼭짓점과의 거리가 호의 반지름) → ③ **벌어질 방향**을 클릭하면 완성됩니다.
2. 라벨은 기본 θ이며, 더블클릭 또는 인스펙터에서 바꿉니다. `Ctrl`을 누르면 각 클릭이 15° 단위로 스냅됩니다.

**직각 표시 (Shift+A)** — 직각 기호(⌐ 모양)를 그립니다.

1. `Shift+A`를 누르고, ① 직각의 **꼭짓점** → ② **한 변 방향의 점**(거리가 기호 크기) → ③ 기호가 **열릴 쪽**을 차례로 클릭합니다.

| 키 | 동작 |
|---|---|
| `A` | 각도 호 |
| `Shift+A` | 직각 표시 |
| `Tab` | 두 도구가 켜진 상태에서 서로 전환 |
| `Ctrl` + 클릭 | 15° 각도 스냅 |

### 2.11 화면 눈금자와 안내선

캔버스 위쪽·왼쪽 가장자리의 **화면 눈금자**(mm 단위)는 도구가 아니라 상시 표시되는 좌표 자입니다. 이를 이용해 **안내선**(인쇄되지 않는 파란 기준선)을 만들 수 있습니다.

1. 위 눈금자를 **클릭**하면 세로 안내선, 왼쪽 눈금자를 클릭하면 가로 안내선이 그 위치에 생깁니다.
2. 안내선은 드래그해서 옮길 수 있습니다. 캔버스 위에서는 객체가 없는 빈 지점에서만 잡힙니다(객체 클릭이 항상 우선).
3. 안내선을 선택하면 눈금자에 좌표 배지(`X = 30.0 mm` 형태)가 나타나고, **배지를 클릭하면 좌표를 숫자로 직접 입력**할 수 있습니다 (`Enter` 확정, `Esc` 취소).
4. 선택된 안내선은 `Delete`로 지웁니다.

| 조작 | 동작 |
|---|---|
| 눈금자 클릭 | 안내선 추가 |
| 안내선 드래그 | 위치 이동 |
| 좌표 배지 클릭 | mm 단위 정확한 위치 입력 |
| `Delete` | 선택된 안내선 삭제 |

**팁** — 여러 문항 그림의 기준선(지면·수면·축)을 같은 높이에 맞출 때, 먼저 안내선을 깔고 그 위에서 그리면 일관된 배치를 얻기 쉽습니다.

## 3. 텍스트와 수식 — 입력·편집·LaTeX 팔레트·라벨

이 장에서는 그림 위에 글자를 올리는 모든 방법을 다룹니다. 5E에서 "글자"는 크게 세 갈래입니다.

| 갈래 | 무엇에 쓰나 | 만드는 방법 |
|---|---|---|
| **텍스트 객체** | 설명문, 물체 이름, 보기 문장 등 자유 배치 글자 | 텍스트 도구(T) |
| **수식 객체** | 분수·첨자·그리스 문자가 들어간 식 (예: v² = v₀² + 2as) | 텍스트 도구에서 LaTeX 문법으로 입력하면 자동 전환 |
| **라벨** | 특정 객체에 붙어 다니는 글자 (지시선 라벨, 각도 기호 θ, 도형·축 이름) | 라벨러 도구(Shift+T), 또는 인스펙터의 라벨 입력란 |

모든 텍스트·수식·라벨의 글자색은 검정으로 고정되어 있습니다(시험지 인쇄 기준). 색을 바꾸는 설정은 없습니다.

---

### 3.1 텍스트 입력 — 텍스트 도구 (T)

캔버스의 원하는 위치에 여러 줄 글자를 놓는 기본 도구입니다.

**사용 절차**

1. 왼쪽 도구막대에서 **텍스트(T)** 버튼을 누르거나 키보드에서 **T** 키를 누릅니다.
2. 글자를 놓을 캔버스 위치를 클릭합니다. 화면 가운데에 **"텍스트 입력"** 창이 뜹니다.
3. 입력칸에 내용을 타이핑합니다. **Enter**를 누르면 줄이 바뀝니다(여러 줄 가능).
4. 위쪽 **미리보기** 영역에서 실제로 그려질 모습을 확인합니다.
5. **확인** 버튼 또는 **Ctrl+Enter**로 확정합니다. 확정하면 자동으로 선택 도구(V)로 돌아옵니다.

**단축키**

| 키 | 동작 |
|---|---|
| T | 텍스트 도구 선택 |
| Enter | 줄바꿈 (입력창 안에서) |
| Ctrl+Enter (또는 ⌘+Enter) | 입력 확정 |
| Esc | 취소 (아무것도 만들지 않음) |

**팁·주의**

- 입력창이 그림을 가리면 창의 **제목줄을 드래그**해서 옮길 수 있습니다.
- 내용을 비운 채 확인하면 텍스트가 만들어지지 않습니다.
- 확정 직후 방금 만든 텍스트가 선택된 상태이므로, 바로 드래그해서 위치를 다듬을 수 있습니다.
- 잘못 만들었으면 **Ctrl+Z** 한 번으로 되돌아갑니다.

---

### 3.2 입력창의 서식 컨트롤 — 글꼴·크기·굵게

입력창 안에서 글꼴, 크기, 굵기를 지정하면 **입력 중인 텍스트 전체**에 적용됩니다(글자 일부만 다른 서식으로 바꾸는 기능은 없습니다).

**컨트롤 구성**

| 컨트롤 | 선택지 | 설명 |
|---|---|---|
| 글꼴 | 한글 텍스트(돋움) / 수식 / 라벨 / 명조 / 고딕 | 기본값은 돋움 계열 |
| 크기 | 6 ~ 72 pt 프리셋 | 최소 6pt |
| 굵게 | 켬/끔 토글 | |

**글꼴 다섯 가지의 용도**

- **한글 텍스트(돋움)**: 일반 설명문의 기본 글꼴.
- **수식**: 평가원 시험지풍 수식체. 알파벳은 기울임(이탤릭), **숫자는 정자**로 자동 렌더됩니다. 물리량 기호(m, v, F 등)를 쓸 때 선택하세요. 이 글꼴에는 별도의 기울임 켜기/끄기가 없습니다(자동 처리).
- **라벨**: 객체 라벨과 동일한 신명중명조 정체. 물체 이름(예: "빗면", "수레")에 어울립니다.
- **명조 / 고딕**: 일반 세리프·맑은 고딕 계열.

**팁·주의**

- 미리보기는 확정 후 캔버스에 그려지는 것과 동일한 방식으로 렌더되므로 믿고 보셔도 됩니다.
- 텍스트 안에 "구간 I", "구간 II", "구간 III"처럼 쓰면 그 로마 숫자 부분만 자동으로 Times 계열 세리프 정자로 그려집니다(시험지 관례). 영어 단어 속 I는 바뀌지 않습니다.

---

### 3.3 수식 입력 — LaTeX 문법 자동 인식

텍스트 입력창에 LaTeX식 표기를 쓰면 앱이 자동으로 수식으로 인식해, 확정 시 **수식 객체**로 만들어 줍니다. 별도의 수식 도구를 찾을 필요가 없습니다.

**수식으로 자동 인식되는 조건** — 다음 중 하나라도 있으면 수식이 됩니다.

- 백슬래시 명령: `\frac`, `\theta`, `\sin` 등
- 아래첨자 `_` 또는 위첨자 `^`
- `frac{`, `vec{`, `sqrt{` 구문

**사용 절차**

1. 텍스트 도구(T)로 캔버스를 클릭해 입력창을 엽니다.
2. 글꼴을 **수식**으로 선택합니다(시험지풍 조판을 원할 때).
3. LaTeX 문법으로 식을 입력합니다. 예: `v^2 = v_0^2 + 2as`
4. 미리보기가 수식 조판으로 바뀌는 것을 확인하고 **확인**(또는 Ctrl+Enter)합니다.

**지원 문법 요약** (입력창 오른쪽 위 **? 버튼**을 누르면 같은 표가 팝업으로 나옵니다)

| 종류 | 쓰는 법 | 결과 |
|---|---|---|
| 분수 | `\frac{1}{2}` | 가로줄 있는 세로 분수 |
| 위/아래첨자 | `x^2`, `v_0`, `x^{10}` | 두 자리 이상은 중괄호 필수 |
| 루트 | `\sqrt{2}` | 근호 |
| 벡터 | `\vec{F}` | 글자 위 화살표 |
| 그리스 문자 | `\theta \pi \lambda \mu \Delta \omega` 등 | θ π λ μ Δ ω |
| 함수 | `\sin \cos \tan \log \ln \exp \lim` 등 | **정자(로만)** 로 렌더되어 변수와 구분됨 |
| 기호 | `\pm \times \cdot \to \leq \geq \neq \infty` 등 | ± × · → ≤ ≥ ≠ ∞ |

**팁·주의**

- **괄호 ( )는 문자 그대로** 출력됩니다. 물리식의 `2a(x - x_0)` 같은 표기를 그대로 쓰면 됩니다.
- 중첩이 됩니다: `T = 2\pi \sqrt{\frac{l}{g}}` 처럼 루트 안 분수도 가능합니다.
- `theta`, `pi`처럼 백슬래시 없이 이름만 써도 확정 시 글자(θ, π)로 바뀝니다. 그리스 글자를 직접 붙여넣어도 됩니다.
- `\sin` 없이 그냥 `sin`이라고 쓰면 이탤릭 변수 s·i·n으로 취급됩니다. 함수는 꼭 백슬래시를 붙이세요.
- **일반 문장에 `_`나 `^`를 넣으면 수식으로 잘못 인식**될 수 있으니, 밑줄 문자는 일반 텍스트에 쓰지 마세요.
- 수식은 한 줄 기준으로 조판됩니다. 여러 줄 식은 수식 객체를 여러 개 만들어 세로로 배치하세요.
- 문법이 틀려도 캔버스가 깨지지 않습니다. 잘못된 부분은 입력한 글자 그대로 표시되며, 미리보기에 "수식을 미리볼 수 없습니다."가 뜨면 문법을 점검하세요.

### 3.4 LaTeX 팔레트와 기호 팔레트 — 버튼으로 빠르게 넣기

입력창 아래쪽에는 자주 쓰는 구조·기호를 클릭 한 번으로 넣는 버튼들이 있습니다. 문법을 외우지 않아도 됩니다.

**수식 팔레트** (아래 줄)

| 버튼 | 입력되는 것 | 비고 |
|---|---|---|
| 분수 | `\frac{}{}` | 커서가 첫 번째 중괄호 안에 놓임 |
| 벡터 | `\vec{}` | 〃 |
| 루트 | `\sqrt{}` | 〃 |
| 아래첨자 | `_{}` | 〃 |
| 위첨자 | `^{}` | 〃 |
| θ λ Δ μ π → ← ± × | 해당 LaTeX 명령 (`\theta` 등) | |
| ₀ ₁ ₂ ₃ ² ³ | `_0` `_1` `_2` `_3` `^2` `^3` | |

**기호 팔레트** (위 줄)

| 구역 | 버튼 | 동작 |
|---|---|---|
| 구간 | I, II, III | Times 정자 서체가 **그 글자에만** 입혀진 채 삽입됨. 앞뒤 한글은 원래 글꼴 유지 |
| 원문자 | ㉠ ㉡ ㉢ ㉣ ㉤ | 보기 번호용 일반 문자로 삽입 (수식으로 승격되지 않음) |

**사용 절차**

1. 입력칸에서 넣고 싶은 위치에 커서를 둡니다(범위를 선택해 두면 그 자리가 대체됩니다).
2. 팔레트 버튼을 클릭합니다. 커서와 포커스는 입력칸에 그대로 유지됩니다.
3. 분수·루트 같은 구조 버튼은 커서가 중괄호 안에 들어가므로 바로 내용을 이어서 타이핑하면 됩니다.

**팁·주의**

- 수식 팔레트 버튼을 누르면 그 입력은 수식으로 취급됩니다. 원문자·구간 버튼은 일반 텍스트로 남습니다.
- 입력칸에는 항상 LaTeX 원문이 남습니다. 완성된 모양은 미리보기에서 확인하세요.

---

### 3.5 기존 텍스트·수식 편집

이미 놓인 텍스트·수식은 네 가지 방법으로 다시 편집할 수 있습니다. 어느 방법이든 같은 입력창이 기존 내용·서식을 채운 채 열립니다.

**편집 진입 방법**

| 방법 | 조작 |
|---|---|
| 더블클릭 | 선택 도구(V)로 텍스트·수식을 더블클릭 |
| 재클릭 | 이미 단독 선택된 텍스트·수식을 (드래그 없이) 한 번 더 클릭 |
| 키보드 | 텍스트·수식 하나만 선택된 상태에서 **F2** 또는 **Enter** |
| 우클릭 | 텍스트·수식 위에서 우클릭 → **"텍스트 수정"** |

**사용 절차**

1. 위 방법 중 하나로 입력창을 엽니다.
2. 내용·글꼴·크기·굵게를 수정합니다.
3. **확인** 또는 **Ctrl+Enter**로 확정, **Esc** 또는 **취소**로 원본 유지.

**팁·주의**

- 내용을 전부 지우고 확인해도 **원본이 삭제되지 않고 그대로 유지**됩니다. 지우려면 객체를 선택하고 Delete 키를 쓰세요.
- 편집으로 텍스트 ↔ 수식이 서로 바뀔 수 있습니다. 일반 텍스트에 `\frac{...}` 을 넣으면 수식 객체가 되고, 수식에서 수식 문법을 모두 지우면 일반 텍스트 객체가 됩니다.
- 한 번의 편집은 한 번의 실행 취소(Ctrl+Z) 단위입니다.

**인스펙터에서 서식만 바꾸기** — 텍스트·수식 하나를 선택하면 오른쪽 인스펙터에 **글꼴** 섹션이 나타납니다. 여기서는 입력창을 열지 않고 글꼴(12종), 크기(pt 숫자 입력, 최소 6), 기울임 체크박스를 바로 바꿀 수 있습니다.

### 3.6 라벨러 — 지시선 달린 라벨 (Shift+T)

그림 속 지점을 짧은 지시선으로 가리키고 그 끝에 글자를 다는 도구입니다. "㉠", "빗면", "mgh" 같은 표시에 씁니다.

**사용 절차**

1. 도구막대의 **라벨러** 버튼을 누르거나 **Shift+T**를 누릅니다.
2. **첫 번째 클릭**: 가리킬 지점(그래프·도형 위의 앵커)을 클릭합니다.
3. 마우스를 움직이면 지시선과 라벨 위치가 실시간 미리보기로 따라옵니다. **Ctrl**을 누른 채 움직이면 지시선 각도가 15° 단위로 스냅됩니다.
4. **두 번째 클릭**: 라벨을 놓을 위치를 클릭합니다.
5. 곧바로 **"라벨 텍스트 입력"** 창이 자동으로 열립니다(기본값 ㉠). 내용을 입력하고 확인합니다. 이 창은 3.2~3.4의 텍스트 입력창과 완전히 같아서 글꼴·크기·굵게·기호 팔레트·수식 입력(LaTeX)을 모두 쓸 수 있습니다.

**단축키**

| 키 | 동작 |
|---|---|
| Shift+T | 라벨러 도구 선택 |
| Ctrl (두 번째 클릭 전) | 지시선 각도 15° 스냅 |
| Enter / Ctrl+Enter / Esc | 줄바꿈 / 확정 / 취소 (입력창 안) |

**팁·주의**

- 지시선은 글자에 닿기 직전에 살짝 끊겨 그려지므로 글자를 가리지 않습니다.
- 라벨 내용도 수식이 될 수 있습니다. `\frac{1}{2}mv^2` 같은 식을 라벨로 달 수 있습니다.
- 더블클릭하면 같은 입력창이 다시 열려 재편집됩니다.
- 라벨러를 선택하면 인스펙터에 지시선의 **길이(mm)**·**각도(°)** 입력란이 나타납니다. 값을 바꾸면 앵커(가리키는 지점)는 고정된 채 라벨 쪽 끝만 이동합니다.

---

### 3.7 각도 호의 라벨 (θ 바꾸기)

각도 호(A 키로 생성) 객체에는 기본 라벨 θ가 붙습니다. 이 기호를 α, ㉠, 60° 등 원하는 글자로 바꿀 수 있습니다.

**사용 절차 (두 가지 경로)**

- **더블클릭**: 각도 호를 더블클릭하면 **"각도 라벨/기호 입력"** 작은 창이 열립니다. 열릴 때 기존 내용이 전체 선택되어 있어 바로 타이핑하면 교체됩니다. Ctrl+Enter로 확정, Esc로 취소.
- **인스펙터**: 각도 호를 선택하면 인스펙터에 **라벨** 입력란(짧은 수정용)과 **"라벨 편집..."** 버튼(입력창 열기)이 나타납니다.

**팁·주의**

- 라벨을 비우면 글자만 사라지고 호는 남습니다.
- 도(°) 기호는 자동으로 붙지 않습니다. 필요하면 직접 입력하세요.

---

### 3.8 도형·선·축에 붙는 라벨 (인스펙터)

사각형·타원·좌표축·점·광학기구·회로 소자 등에는 객체에 **붙어 다니는** 라벨을 인스펙터에서 달 수 있습니다. 별도 텍스트 객체와 달리 도형을 옮기면 함께 따라옵니다.

**사용 절차 (사각형·타원 기준)**

1. 도형을 선택합니다.
2. 인스펙터의 **라벨** 칸에 글자를 입력하고 Enter.
3. **위치**를 가운데/위/아래/왼쪽/오른쪽에서 고릅니다.
4. **크기**(pt)로 라벨 글자 크기를 조절합니다.

**객체별 라벨 자리**

| 객체 | 인스펙터 항목 |
|---|---|
| 사각형·타원 | 라벨, 위치(5방향), 크기, 종류 |
| 좌표축 | X 라벨, Y 라벨, 종류 |
| 점(node) | 라벨, 위치(위/아래) |
| 각도 호·광학기구·회로 소자 | 라벨(또는 단자1/단자2), 종류 |

**라벨 "종류" — 물리량 vs 라벨**

여러 객체의 라벨에는 **종류** 드롭다운이 있습니다. 시험지 조판 관례에 맞춰 글꼴이 달라집니다.

| 종류 | 글꼴 | 이럴 때 |
|---|---|---|
| **물리량** | 수식체 이탤릭(숫자는 정자) | m, v, F, T₀ 같은 기호 |
| **라벨** | 신명중명조 정체 | "물체", "빗면", A, ㉠ 같은 이름 |

**팁·주의**

- 도형을 회전시켜도 라벨은 항상 똑바로(정립) 그려집니다.
- 라벨을 비우면 라벨만 사라지고 도형은 그대로입니다.

---

### 3.9 여러 객체의 글씨를 한꺼번에 바꾸기 — 전체 통일/수정

텍스트·라벨이 많아진 뒤 크기나 글씨체를 일괄로 맞추는 기능입니다.

**사용 절차**

1. 바꿀 객체들을 선택합니다(아무것도 선택하지 않으면 **전체**가 대상입니다).
2. 상단 도구 영역의 **"전체 통일/수정"** 버튼을 누릅니다.
3. 모드를 고릅니다: **통일**(모두 같은 값으로) 또는 **증감**(현재 값에서 ±).
4. **글씨 크기**(pt)를 지정하거나, 통일 모드에서 **글씨체**를 고른 뒤 **적용**을 누릅니다.

**팁·주의**

- "글씨 크기"는 텍스트 객체의 크기와 각종 라벨 크기(labelSize를 가진 객체)를 함께 바꿉니다. "글씨체"는 텍스트·수식 객체에만 적용됩니다.
- 적용 전체가 실행 취소(Ctrl+Z) 한 번으로 되돌아갑니다.

## 4. 함수그래프와 좌표평면

5E에서 수학 그래프는 **좌표평면(coordplane)** 과 그 위에 얹히는 **함수 그래프(funcgraph)** 두 객체의 짝으로 이루어집니다. 좌표평면을 따로 만드는 버튼은 없으며, **[함수 입력] 도구 하나**가 두 가지를 함께 처리합니다 — 함수를 입력하면 좌표평면이 없을 경우 자동으로 만들어지고, 이미 선택해 둔 좌표평면이 있으면 그 위에 그래프가 추가됩니다.

---

### 4.1 함수 입력으로 그래프 그리기

수식(예: `sin(5x)`, `x^2-3x+1`)을 입력하면 좌표평면과 곡선을 캔버스에 한 번에 그려 주는 기능입니다.

**사용 절차**

1. 왼쪽 패널의 **공통 도구**에서 **[함수 입력]** 버튼을 누릅니다(또는 키보드 `F`).
2. "함수 입력" 창이 열리고, `y =` 입력칸에 기본값 `sin(x)`가 채워져 있습니다. 원하는 수식으로 바꿔 입력합니다. 입력칸 아래의 도우미 버튼(`sin`, `cos`, `log`, `√`, `π` 등)을 눌러 커서 위치에 끼워 넣을 수도 있습니다.
3. 필요하면 **선 굵기**(0.1~2mm), **선 종류**(실선/점선/파선), **정의역**(그리는 x 구간)을 조정합니다. 오른쪽 미리보기에 결과가 실시간으로 표시됩니다.
4. **[확인]** 을 누릅니다(또는 입력칸에서 `Enter`).
5. 캔버스 화면 가운데에 좌표평면 + 그래프가 삽입되고, 자동으로 선택 도구(V)로 돌아오며 새 그래프가 선택된 상태가 됩니다.

**기존 좌표평면에 그래프를 추가하려면** — 좌표평면을 먼저 클릭해 선택한 뒤 `F`를 누르세요. 미리보기가 그 평면의 범위·격자 설정을 그대로 물려받고, 확인 시 새 평면을 만들지 않고 그 평면 위에 그려집니다.

**단축키·조작**

| 키/조작 | 동작 |
|---|---|
| `F` | 함수 입력 창 열기 |
| `Enter` (수식 입력칸에서) | 확인(삽입) |
| `Esc` / 창 바깥 클릭 / [취소] | 창 닫기 |
| `Ctrl+Z` | 삽입 취소(평면+그래프가 한 번에 되돌아감) |
| `Ctrl+F` → "함수" 검색 | 오브젝트 검색으로 도구 찾기 |

**팁·주의**

- 삽입은 **실행 취소 1회분**으로 기록됩니다. 새로 만들어진 평면과 그래프가 `Ctrl+Z` 한 번에 함께 사라집니다.
- **삼각형이 선택된 상태에서는 `F`가 동작하지 않습니다.** 그때의 `F`는 삼각형 상하 뒤집기 단축키입니다. 빈 곳을 클릭해 선택을 해제한 뒤 누르세요.
- 수식이 잘못되면 확인 버튼이 비활성화되고 미리보기 옆에 빨간 오류 메시지(예: `알 수 없는 이름: "..."`)가 표시됩니다.

---

### 4.2 수식 문법

함수 입력 창이 알아듣는 수식 규칙입니다. 계산기가 아니라 **허용 목록 방식**이라, 목록에 없는 이름을 쓰면 즉시 오류로 알려 줍니다.

**쓸 수 있는 요소**

| 구분 | 내용 |
|---|---|
| 변수 | `x` (대소문자 무관) |
| 연산 | `+` `-` `*` `/` `^`(거듭제곱) |
| 삼각·역삼각 | `sin` `cos` `tan` `asin` `acos` `atan` |
| 쌍곡 | `sinh` `cosh` `tanh` |
| 로그·지수 | `log`(**상용로그**, 밑 10) `ln`(자연로그) `exp` |
| 기타 함수 | `sqrt` `abs` `floor` `ceil` `round` `sign` |
| 상수 | `pi`(π), `e` |

**규칙**

1. **곱셈 기호는 생략 가능** — `5x`, `2(x+1)`, `5sin(x)`, `(x+1)(x-1)` 모두 자동으로 곱셈으로 인식됩니다.
2. **함수에는 괄호가 필수** — `sin(5x)` (○), `sin5x` (×, 오류로 표시됨).
3. **각도는 라디안**입니다. 도(°) 단위 입력은 지원하지 않으므로 `sin(pi*x/180)`처럼 직접 환산하세요.
4. 정의되지 않는 구간(예: `log(x)`의 x≤0, `1/x`의 x=0 부근)은 **곡선이 자연스럽게 끊어져** 그려집니다. 극(pole)을 가로지르는 세로선이 생기지 않습니다.

---

### 4.3 여러 함수를 한 평면에 넣기

함수 입력 창 한 번으로 여러 개의 함수를 같은 좌표평면에 겹쳐 그리는 기능입니다.

**사용 절차**

1. 함수 입력 창 상단의 **[+ 함수 추가]** 를 누르면 함수 칩(chip)이 하나 늘어납니다.
2. 칩을 클릭하면 그 함수가 편집 대상이 되고, 아래의 수식·선 굵기·선 종류·정의역 컨트롤이 그 함수의 값으로 바뀝니다. 미리보기에서 **선택된 함수만 파란색**으로 강조됩니다.
3. 함수가 2개 이상일 때 칩의 `×`를 누르면 그 함수가 목록에서 삭제됩니다.
4. **[확인]** 을 누르면 모든 함수가 한 평면에 한 번에 삽입되고(실행 취소 1회분), 삽입된 그래프 전체가 선택 상태가 됩니다.

**단축키·조작**

| 조작 | 동작 |
|---|---|
| 칩 클릭 | 그 함수를 편집 대상으로 선택 |
| 칩의 `×` 클릭 | 그 함수 삭제(2개 이상일 때만 표시) |
| [+ 함수 추가] | 빈 함수 추가 후 바로 입력 |

**팁·주의**

- 5E는 회색조 출력이 기본이라 **함수 구분은 색이 아니라 선 종류(실선/점선/파선)** 로 합니다. 시험지 인쇄를 생각하면 이 방식이 안전합니다.
- 여러 함수 중 하나라도 수식 오류가 있으면 전체가 삽입되지 않고 `수식: 오류내용` 형태로 어느 함수가 문제인지 알려 줍니다.

---

### 4.4 정의역(그리는 구간) 지정

함수를 평면 전체가 아니라 원하는 x 구간에서만 그리는 기능입니다.

**사용 절차**

1. 함수 입력 창에서 **정의역** 숫자칸 두 개(최소 ~ 최대)에 값을 직접 입력하거나,
2. 미리보기 안의 **파란 세로 핸들 두 개**를 좌우로 드래그합니다. 정의역 밖 구역은 회색 음영으로 표시됩니다.
3. 삽입 후에도 그래프를 선택하면 인스펙터의 "함수 그래프" 섹션에서 정의역을 다시 고칠 수 있습니다(4.7절).

**조작**

| 조작 | 동작 |
|---|---|
| 세로 핸들 드래그 | 선택된 함수의 정의역 경계 이동 |
| 숫자칸 입력 | 정의역 직접 지정(0.5 단위 스피너) |

**팁·주의**

- 정의역은 **평면의 x 범위 안으로 자동 제한**됩니다. 평면이 -5~5이면 정의역도 그 안에서만 정할 수 있습니다.
- `x^2`처럼 값이 빠르게 커지는 함수는 **y 범위를 벗어난 부분이 평면 상자 밖까지 그려집니다.** 이럴 땐 정의역을 좁히거나(예: -2~2), 평면의 y 범위를 넓히세요(4.6절).

---

### 4.5 좌표평면 기본 조작

좌표평면은 축·격자·눈금·라벨을 갖춘 상자로, 함수 그래프의 "모눈종이" 역할을 합니다.

**자동 생성되는 기본 사양**

| 항목 | 기본값 |
|---|---|
| 범위 | x, y 모두 -5 ~ 5 |
| 눈금 간격 | 1 |
| 크기 | 48×48mm (1칸 = 4.8mm 정사각형) |
| 형태 | 십자(4사분면) |
| 표시 | 축선 ○ · 격자 ○ · 눈금 ○ · 숫자 라벨 × |
| 라벨 | x축 `x`, y축 `y`, 원점 `O` |

**사용 절차(선택·이동·크기)**

1. 좌표평면은 선 위가 아니라 **상자 내부 아무 곳이나** 클릭해도 선택됩니다.
2. 드래그로 이동하고, 모서리 핸들로 크기를 조절합니다. 기본적으로 **정사각 비율이 잠겨 있어** 칸이 늘 정사각형으로 유지됩니다.
3. 형태는 인스펙터 또는 상세 편집에서 **십자**(4사분면) / **L자**(1사분면, 평가원 양식) / **직선**(x축만, 세로축 없음) 중에 고릅니다.

**단축키·조작**

| 키/조작 | 동작 |
|---|---|
| 클릭 | 선택 |
| **더블클릭** | 상세 편집 창 열기(4.6절) |
| 방향키 | 0.5mm 이동 (`Ctrl`+방향키 = 5mm) |
| `Delete` | 삭제 |
| `Shift`+클릭 / 빈 곳 드래그(마퀴) | 여러 객체 함께 선택 |

**팁·주의**

- **평면만 옮기면 그래프는 따라오지 않습니다.** 평면과 그래프를 함께 옮길 때는 빈 곳에서 드래그(마퀴)로 둘 다 감싸 선택한 뒤 이동하세요.
- 평면의 범위·크기를 나중에 바꿔도 **이미 그려진 그래프는 자동으로 다시 계산되지 않습니다.** 그래프를 선택해 인스펙터에서 정의역이나 수식을 한 번 다시 확정하면 현재 평면 기준으로 재계산되어 제자리를 찾습니다.
- 빈 좌표평면만 필요하면: 함수 입력으로 아무 함수나 넣은 뒤, **그래프만 선택해 `Delete`** 하면 평면이 남습니다.

---

### 4.6 좌표평면 상세 편집 창

설정 항목이 많은 좌표평면을 실시간 미리보기와 함께 한 화면에서 고치는 전용 창입니다.

**사용 절차**

1. 좌표평면을 **더블클릭**하거나, 선택 후 오른쪽 인스펙터의 **[상세 편집…]** 버튼을 누릅니다.
2. 왼쪽에서 항목을 고치면 오른쪽 미리보기에 즉시 반영됩니다(실제 캔버스는 아직 그대로).
3. **[확인]** 을 누르면 모든 변경이 한 번에 적용됩니다(실행 취소 1회분). [취소]·`Esc`·바깥 클릭은 변경을 버립니다.

**편집 항목**

| 항목 | 내용 |
|---|---|
| 형태 | 십자 / L자 / 직선 |
| x·y 범위 | 표시할 수학 좌표 구간 |
| 눈금 간격 | x·y 각각의 격자/눈금 간격 |
| 표시 | 축선 · 격자 · 눈금 · 숫자 라벨 각각 켜기/끄기 |
| 숫자 크기 | 숫자 라벨 글자 크기(mm) — 숫자 라벨이 켜져 있을 때만 표시 |
| x축·y축 이름 | 축 끝의 라벨 문자(예: `t`, `v`) |
| 원점 라벨 | 체크로 표시 여부, 칸에 글자 지정(**비우면 원점 글자 숨김**) |
| 이름 크기 | 축 이름 표시 여부 + 글자 크기(mm) |
| 내보내기 | 체크 해제 시 이 평면은 SVG/PNG 내보내기에서 제외(밑그림·보조선 용도) |

**단축키·조작**

| 키/조작 | 동작 |
|---|---|
| 좌표평면 더블클릭 | 창 열기 |
| `Esc` / 바깥 클릭 / [취소] | 변경 버리고 닫기 |
| [확인] | 일괄 적용(실행 취소 1회) |

**팁·주의**

- **격자는 범위보다 한 칸 안쪽까지만** 그려집니다(범위 -5~5면 격자는 -4~4). 축 끝 화살표 주변을 깔끔하게 비우는 평가원 양식 규칙이라 고장이 아닙니다. 숫자 라벨도 0과 범위 끝값은 생략됩니다.
- 축 이름·원점 라벨은 입력한 문자가 수식 글꼴(이탤릭)로 그대로 표시됩니다. 창 안내문에 LaTeX 문법(`v_0`, `\theta`)이 언급되어 있으나, 현재 버전에서는 이런 명령이 기호로 변환되지 않을 수 있으므로 **그리스 문자가 필요하면 `θ` 같은 유니코드 문자를 직접 붙여넣고 미리보기로 확인**하는 것이 안전합니다.
- "직선" 형태에서는 세로축·가로 격자가 함께 사라집니다. 시간축 하나만 필요한 그림에 적합합니다.

### 4.7 삽입한 그래프 고치기 (인스펙터)

캔버스의 함수 그래프를 선택하면 오른쪽 인스펙터에 "함수 그래프" 섹션이 나타나며, 수식·정의역·선 모양을 수정할 수 있습니다.

**사용 절차**

1. 그래프 곡선을 클릭해 선택합니다.
2. **수식** 줄의 **[편집]** 버튼을 누르면 입력창이 뜹니다. 수식을 고치고 확인하면 소속 평면 기준으로 곡선이 다시 계산됩니다. (문법 오류면 원래 수식으로 되돌아가고 오류가 표시됩니다.)
3. **정의역** 두 칸으로 그리는 구간을 조정합니다 — 값을 바꾸는 즉시 재계산됩니다.
4. **선 종류**(실선/점선1/점선2/점선3)를 고릅니다. 선 굵기와 명암(회색 단계)은 공용 "선" 섹션에서 조절합니다.

**곡선으로 변환** — 수식 구동을 끊고 점을 직접 편집할 수 있는 일반 곡선으로 바꾸는 버튼입니다. 그래프 일부만 남기거나 손으로 모양을 다듬고 싶을 때 씁니다.

1. 그래프 선택 → 인스펙터의 **[곡선으로 변환]** 클릭.
2. 이후 일반 곡선처럼 점 편집·채우기 등이 가능해지는 대신, **수식·정의역 편집은 더 이상 할 수 없습니다.**

**단축키·조작**

| 키/조작 | 동작 |
|---|---|
| 곡선 클릭 | 그래프 선택(인스펙터 표시) |
| `Ctrl+Z` / `Ctrl+Shift+Z` | 실행 취소 / 다시 실행 |
| `Delete` | 그래프만 삭제(평면은 남음) |

**팁·주의**

- 곡선으로의 변환은 **일방통행**입니다. 다시 함수 그래프로 되돌리는 기능은 없으며, 변환 직후라면 `Ctrl+Z`로만 복구할 수 있습니다.
- 그래프가 딸린 좌표평면을 지운 경우, 그 그래프의 수식·정의역을 편집하려 하면 "소속 좌표평면을 찾을 수 없습니다"라는 안내가 나옵니다. 이때는 [곡선으로 변환]으로 일반 곡선으로 바꿔 계속 쓰면 됩니다.
- 잠금(자물쇠)된 그래프는 모든 편집 버튼이 비활성화됩니다.

## 5. 이미지 작업 — 붙여넣기·벡터화·객체화·오려내기·선 추출·비교

교과서 그림, 기출 문항, 손으로 그린 스케치 같은 **이미지(사진) 파일**을 캔버스로 가져와 활용하는 방법을 다룹니다. 5E에서 이미지는 두 가지 방식으로 쓸 수 있습니다.

- **이미지 객체 그대로 쓰기** — 사진처럼 통째로 배치하고 크기·투명도만 조절합니다. 밑그림(트레이싱 배경)으로 깔고 그 위에 따라 그릴 때 좋습니다.
- **객체화해서 쓰기** — 그림 속 선과 면을 분석해 5E의 편집 가능한 객체(선·곡선·다각형 등)로 분리합니다. 변환 후에는 색·굵기·모양을 자유롭게 고칠 수 있습니다.

---

### 5.1 이미지 가져오기 — 붙여넣기·불러오기·끌어놓기

**무엇인가:** PNG/JPG 이미지를 캔버스에 이미지 객체로 삽입하는 네 가지 입구입니다.

#### 방법 A — 클립보드 붙여넣기 (가장 빠름)

1. 캡처 도구(예: `Win+Shift+S`)로 화면을 캡처하거나, 다른 프로그램에서 이미지를 복사합니다.
2. 5E 캔버스로 돌아와 마우스를 **이미지를 놓고 싶은 자리**에 둡니다.
3. `Ctrl+V`를 누릅니다. 마우스 위치를 중심으로 이미지가 삽입되고, 곧바로 선택된 상태가 됩니다.

- 이미지는 아트보드의 90% 안에 들어오도록 자동 축소되며, 가로세로 비율이 고정된 채 들어옵니다.
- **주의:** 5E 안에서 `Ctrl+C`로 오브젝트를 복사해 둔 상태라면 `Ctrl+V`는 그 오브젝트 붙여넣기가 우선합니다. 이때는 시스템 클립보드의 이미지가 붙지 않습니다.

#### 방법 B — 파일 메뉴에서 불러오기

1. 상단 **파일 ▾ → 이미지 불러오기**를 클릭합니다.
2. PNG 또는 JPG 파일을 선택합니다. 이미지가 아트보드에 배치되고 "원하는 크기로 조정이 완료되면 Enter를 눌러주세요" 안내가 나타납니다.
3. 핸들로 크기를 조절하고 원하는 위치로 옮긴 뒤 `Enter`로 확정합니다. 마음이 바뀌면 `Esc` — 이미지가 삭제되며 취소됩니다.

#### 방법 C — 탐색기에서 끌어놓기

1. 윈도우 탐색기에서 이미지 파일을 캔버스 위로 끌어다 놓습니다.
2. 놓은 지점을 중심으로 배치되고, 방법 B와 같은 `Enter` 확정 단계가 이어집니다.

#### 방법 D — 기출문항 검색에서 삽입

1. 캔버스 하단의 **기출문항 검색** 버튼(또는 `Ctrl+Shift+F`)으로 검색 창을 엽니다.
2. 과목을 먼저 선택한 뒤 검색합니다. 번호 검색(예: `261101` = 2026학년도 수능 1번)과 해시태그 검색(예: `#역학#도르레`)을 지원합니다.
3. 카드(문항)를 클릭해 선택하고 **[이미지 삽입]**을 누르면 방법 A와 같은 방식으로 캔버스에 들어옵니다. (**[오브젝트 변환]**은 5.3의 객체화 창으로 바로 보냅니다.)

| 키/조작 | 동작 |
|---|---|
| `Ctrl+V` | 클립보드 이미지 붙여넣기 (마우스 위치 중심) |
| `Enter` | (불러오기·끌어놓기 배치 중) 배치 확정 |
| `Esc` | (배치 중) 취소 — 이미지 제거 |
| `Ctrl+Shift+F` | 기출문항 검색 창 열기 |
| `Delete` | 선택한 이미지 삭제 |

**팁·주의**
- 붙여넣기(`Ctrl+V`)는 `Enter` 확정 단계 없이 바로 들어옵니다. 크기·위치는 삽입 후 선택 도구로 조절하면 됩니다.
- 붙여넣기·불러오기는 PNG/JPG만 받습니다. (객체화 창은 WEBP도 받습니다 — 5.3 참고.)
- 삽입된 이미지는 프로젝트 파일(JSON) 안에 통째로 저장되므로, 큰 이미지를 여러 장 넣으면 프로젝트 파일이 커집니다.

---

### 5.2 이미지 관리 패널 — 투명도·선택금지·위치고정

**무엇인가:** 우측 속성 패널(인스펙터) 맨 위에 항상 떠 있는 **'이미지' 섹션**으로, 캔버스의 모든 이미지를 한곳에서 관리합니다. 이미지가 없으면 "이미지 없음"으로 표시됩니다.

1. 인스펙터 맨 위 **이미지** 섹션을 펼칩니다.
2. **이미지 선택** 드롭다운에서 관리할 이미지를 고릅니다. (선택금지가 아닌 이미지는 캔버스에서도 함께 선택됩니다.)
3. 아래 항목으로 상태를 조절합니다.

| 항목 | 동작 |
|---|---|
| 투명도 | 0~100% 슬라이더. 밑그림용으로 흐리게 깔 때 사용 |
| 선택금지 | 체크하면 캔버스 클릭으로 선택되지 않음 — 위에 그림을 그려도 이미지가 걸리적거리지 않음 |
| 위치고정 | 체크하면 이동 잠금 (선택금지 상태에서는 비활성) |
| 비교 | 원본과 내가 그린 그림을 나란히 비교 (5.6 참고) |

**팁·주의 — 따라 그리기(트레이싱) 기본 순서**
1. 이미지를 붙여넣고 → 2. 투명도를 30~50%로 낮추고 → 3. **선택금지**를 켠 뒤 → 4. 그 위에 도구로 따라 그리고 → 5. **비교**로 결과를 확인하고 → 6. 다 그렸으면 선택금지를 해제(해제하면 그 이미지가 자동 선택됨)하고 `Delete`로 지웁니다.

---

### 5.3 이미지 객체화(벡터화) — 그림을 편집 가능한 객체로

**무엇인가:** 이미지 속에서 서로 떨어져 있는 잉크 덩어리를 자동으로 찾아, 각각을 5E에서 편집 가능한 개별 객체(다각형·곡선 등)로 분리해 삽입하는 기능입니다. 흰 배경은 자동으로 투명 처리됩니다.

1. 좌측 도구 패널 **고급 기능 → [이미지 객체화]** 버튼을 누릅니다. (`Ctrl+T`도 되지만, 일부 브라우저는 이 키를 '새 탭'으로 가로챕니다 — 그때는 버튼을 쓰세요. 기출문항 검색의 **[오브젝트 변환]**으로도 열립니다.)
2. 이미지를 넣습니다 — 창 안으로 **끌어놓기**, 안내 영역 **클릭 후 파일 선택**, 또는 `Ctrl+V` **붙여넣기**. PNG/JPG/WEBP를 지원하며, 2000px보다 큰 이미지는 처리용으로 자동 축소됩니다.
3. 자동으로 분석이 실행되어 미리보기에 인식 결과가 색으로 표시됩니다 — **파랑 = 도형, 빨강 = 글자 추정**.
4. 미리보기를 점검합니다: 휠로 확대/축소, 빈 곳 드래그로 이동, **원치 않는 조각은 클릭해 제외**(다시 클릭하면 포함). [전체 보기]로 초기 화면으로 돌아옵니다.
5. 인식이 마음에 들지 않으면 우측 설정을 조절합니다. 슬라이더를 움직이면 잠시 후 자동으로 다시 분석됩니다.
6. **글자(라벨) 처리** 방식을 고릅니다 (아래 표).
7. **[객체로 삽입]**을 누릅니다. 분리된 객체들이 아트보드 중앙(아트보드의 80% 크기 이내)에 삽입되고 전부 선택된 상태가 됩니다.

**설정 항목**

| 설정 | 기본값 | 의미 |
|---|---|---|
| 오브젝트 묶음 거리 | 3px (1~9) | 가까운 잉크를 하나의 객체로 묶는 거리. 키우면 덩어리가 크게 묶임 |
| 최소 오브젝트 크기 | 25px² (5~400) | 이보다 작은 조각(점·잡티)은 무시 |
| 글자 판정 크기 기준 | 22px (8~60) | 이 크기 이하의 조각을 글자로 추정 |
| 곡선 단순화 정도 | 1.2 (0~4.0) | 키울수록 점 개수가 줄어 매끈해지지만 세부가 뭉개짐 |
| 회색 단계 보존 | 켜짐 | 흰/회색/검정 다단계 톤을 살려서 인식 (끄면 흑백 2단계) |
| 격자·눈금선 제거 | 꺼짐 | 그래프·도표의 긴 격자선을 지우고 곡선만 남김 |
| 원본 이미지를 반투명 배경으로 함께 삽입 | 꺼짐 | 검수용 원본을 투명도 28%로 깔아줌 (내보내기 제외·선택금지 상태로 삽입) |
| [고급] 선·도형 승격 | 꺼짐 | 5.4 참고 (실험 기능) |

**글자(라벨) 처리 방식**

| 방식 | 결과 |
|---|---|
| 원본 이미지로 유지 (권장·기본) | 글자 부분만 잘라 투명 배경의 작은 이미지로 삽입 — 모양 손실 없음 |
| 남기기 | 글자 획을 도형(폴리곤)으로 그대로 변환 |
| 지우기 | 글자 조각을 삽입하지 않음 |
| 텍스트 객체로 대체 (A, B, C…) | 글자 자리에 순번 라벨(수식 객체)을 넣음 — 라벨을 새로 달 때 편리 |

| 키/조작 (객체화 창 안) | 동작 |
|---|---|
| 휠 | 미리보기 확대/축소 (커서 기준) |
| 드래그 | 미리보기 이동 |
| 조각 클릭 | 제외 ↔ 포함 전환 (제외된 조각은 회색 점선 테두리) |
| `Ctrl+V` | 창에 이미지 붙여넣기 (교체 가능) |
| `Esc` | 창 닫기 |

**팁·주의**
- 배경이 흰색(또는 밝은 단색)인 그림에 최적화되어 있습니다. 사진처럼 복잡한 이미지는 객체화보다 이미지 그대로 쓰는 편이 낫습니다.
- 삽입은 실행취소 1단계로 처리됩니다 — 결과가 마음에 안 들면 `Ctrl+Z` 한 번으로 전부 되돌립니다.
- 상태줄에 "오브젝트 N개 (글자 추정 M개) · 제외 K개"가 표시되니 삽입 전에 개수를 확인하세요. 조각이 너무 많으면 '최소 오브젝트 크기'를 키우세요.

### 5.4 선 추출 — '선·도형 승격' 옵션 (실험 기능)

**무엇인가:** 객체화 과정에서 **가는 획을 면이 아닌 진짜 '선 객체'로**, 반듯한 사각형·원을 '상자'·'원' 객체로 승격해 주는 고급 옵션입니다. 변환 후 선 굵기·끝모양 등을 도형답게 편집할 수 있게 됩니다.

1. 객체화 창(5.3)에서 **[고급·미완성 ⚠] 선·도형 승격** 체크박스를 켭니다. 자동으로 다시 분석됩니다.
2. 미리보기의 색으로 판정 결과를 확인합니다 (아래 범례).
3. 잘못 판정된 조각은 클릭해 제외하거나, 슬라이더를 조정해 재분석합니다.
4. **[객체로 삽입]** — 획은 선/꺾은선/곡선 객체로, 상자·원은 해당 도형 객체로 들어옵니다.

**미리보기 범례 (옵션 켰을 때)**

| 색 | 판정 |
|---|---|
| 파랑 ■ | 선(가는 획) |
| 초록 ■ | 상자(사각형) |
| 청록 ■ | 균일 띠 (테두리+채움 통합) |
| 보라 ■ | 원/타원 |
| 주황 ■ | 글자 추정 |

**팁·주의**
- 대략 **8픽셀 이하의 가늘고 굵기가 일정한 획**만 선으로 승격됩니다. 조건에 맞지 않으면 자동으로 일반 조각(면 채움 다각형)으로 처리되므로 결과물이 사라지지는 않습니다.
- 이름 그대로 **실험 기능**입니다(버튼에도 "미완성 ⚠" 표시). 결과가 부정확할 수 있으니 평소에는 꺼 두고, 그래프·회로도처럼 선 위주의 그림에서만 시도해 보세요.
- 화살촉처럼 획으로 설명되지 않는 부분은 별도 조각으로 함께 삽입됩니다.

### 5.5 오려내기(영역 지우기) — 현재 준비 중

**무엇인가:** 이미지의 일부 영역(사각형 또는 자유 곡선으로 둘러싼 부분)을 투명하게 지워내는 기능입니다. 문항 이미지에서 필요 없는 보기나 번호만 도려낼 때 쓰도록 설계되었습니다.

**현재 버전(v0.54.14)에서는 이 기능의 진입 버튼이 화면에 노출되지 않아 사용할 수 없습니다.** 내부적으로는 "사각형 영역 지우기 / 자유 영역 지우기 / 지운 영역 초기화" 도구와, 드래그로 영역을 그린 뒤 `Enter`로 확정하고 `Esc`로 취소하는 동작까지 구현되어 있으므로, 이후 버전에서 다시 열릴 예정입니다. 그때까지는 다음으로 대신하세요.

- 이미지의 불필요한 부분 제거 → 삽입 전에 캡처 도구로 필요한 부분만 캡처해 붙여넣기
- 이미지 일부만 객체로 쓰기 → 객체화(5.3)에서 원치 않는 조각을 클릭해 제외

### 5.6 비교 — 원본 이미지와 내 그림 나란히 보기

**무엇인가:** 이미지를 흐리게 깔고 따라 그린 뒤, **원본(불투명)과 내가 그린 오브젝트만**을 같은 영역·같은 배율로 좌우에 나란히 띄워 얼마나 닮았는지 확인하는 팝업입니다. 순수 표시 기능이라 저장·실행취소·내보내기에 아무 흔적도 남기지 않습니다.

1. 인스펙터 상단 **이미지** 패널에서 비교할 원본 이미지를 '이미지 선택'으로 고릅니다.
2. **[비교]** 버튼을 누릅니다. 화면이 어두워지고 "비교할 영역을 드래그하십시오" 안내가 뜹니다.
3. 비교하고 싶은 영역을 드래그해 사각형을 그립니다.
4. 필요하면 8개 핸들로 크기를 조절하거나, 사각형 본체를 끌어 옮기거나, 크기 라벨을 클릭해 mm 수치를 직접 입력합니다.
5. `Enter`를 누르면 팝업이 열립니다 — **왼쪽 = 삽입한 원본 이미지(투명도와 무관하게 원본 그대로), 오른쪽 = 내가 그린 오브젝트**(이미지는 모두 제외). 영역의 실제 크기(mm)도 함께 표시됩니다.
6. [닫기] 버튼, `Esc`, 또는 팝업 바깥 클릭으로 닫습니다.

| 키/조작 (영역 지정 중) | 동작 |
|---|---|
| 드래그 | 비교 영역 그리기 / (그린 뒤) 본체 이동 |
| 핸들 드래그 | 영역 크기 조절 |
| 크기 라벨 클릭 | 가로 mm 직접 입력 → `Tab` → 세로 입력 |
| `Enter` | 영역 확정 → 비교 팝업 열기 |
| `Esc` 또는 우클릭 | 취소 |

**팁·주의**
- 트레이싱을 위해 투명도를 낮춰 두었어도 왼쪽에는 항상 **원본 그대로(불투명)** 나오므로, 캔버스의 투명도 설정을 되돌릴 필요가 없습니다.
- 오른쪽에는 모든 이미지가 빠지고 순수하게 내가 그린 오브젝트만 표시됩니다.
- 원본 이미지를 회전시켜 둔 경우에는 왼쪽이 정확한 부분 확대 대신 원본 전체의 근사 표시로 대체됩니다 — 비교는 회전 전에 하는 것이 정확합니다.

## 6. 과학 오브젝트 라이브러리 — 물리·화학·생물·지구과학·퍼스널 저장소

과학 문항 도해에 자주 쓰는 기호와 실험 기구(회로 소자, 렌즈·거울, 도르래, 단진자 등)를 버튼 한 번으로 캔버스에 올릴 수 있게 모아 둔 것이 **과학 오브젝트 라이브러리**입니다. 화면 **왼쪽 도구 패널**이 그 무대이며, 위에서부터 다음 순서로 배치되어 있습니다.

1. **과목 선택** 드롭다운 (물리학 / 화학 / 생명과학 / 지구과학)
2. **공통 도구** — 선택·도형·선·텍스트 등 (5장 참고)
3. **과목별 오브젝트** — 선택한 과목의 파트별 심볼 라이브러리
4. **퍼스널 오브젝트** — 내가 만든 도해를 저장해 두고 다시 쓰는 개인 저장소
5. **고급 기능** — 이 장에서는 그중 [오브젝트 저장] 버튼을 사용합니다

---

### 6.1 과목 선택 — 라이브러리와 화면 테마를 함께 전환

왼쪽 패널 맨 위의 드롭다운에서 과목을 고르면, 화면의 강조색 테마와 '과목별 오브젝트' 목록, 그리고 퍼스널 오브젝트에 보이는 항목이 그 과목에 맞게 한꺼번에 바뀝니다.

**사용 절차**

1. 왼쪽 도구 패널 맨 위의 **과목 선택** 드롭다운을 클릭합니다.
2. **물리학 / 화학 / 생명과학 / 지구과학** 중 하나를 선택합니다.
3. 화면 강조색이 과목 색으로 바뀌고, 아래 '과목별 오브젝트' 섹션의 파트 목록이 해당 과목의 것으로 교체됩니다.

| 조작 | 동작 |
|---|---|
| 드롭다운 선택 | 과목 전환 (강조색 테마 + 라이브러리 + 퍼스널 오브젝트 필터 동시 전환) |
| (자동) | 선택한 과목은 브라우저에 기억되어 다음 실행 때 그대로 복원됩니다 |

**팁·주의사항**

- 과목 선택은 퍼스널 오브젝트에도 영향을 줍니다. 퍼스널 오브젝트는 **저장 당시의 과목에서만 보이므로**(6.4절), "저장했는데 목록에 없다" 싶으면 먼저 과목이 맞는지 확인하세요.
- 과목 선택 값은 '설정 저장하기' 파일에도 포함되어 다른 컴퓨터로 옮길 수 있습니다(6.4.5절).

---

### 6.2 과목별 오브젝트 — 파트별 심볼 라이브러리

선택한 과목의 단원(파트)별로 심볼 버튼이 아코디언(접이식 목록)으로 정리되어 있습니다.

**사용 절차**

1. 왼쪽 패널에서 **과목별 오브젝트** 섹션 제목을 클릭해 펼칩니다. (기본은 접힌 상태)
2. 원하는 파트(예: 물리학 → **전기자기학**) 제목을 클릭해 펼칩니다.
3. 심볼 버튼을 클릭하면 버튼이 강조색으로 활성화되고, 캔버스에서 클릭 또는 드래그로 배치합니다(방식은 아래 표 참고).
4. 배치가 끝나면 자동으로 **선택(V) 도구**로 돌아옵니다.

현재 버전(v0.54.14)의 파트 구성은 다음과 같습니다.

| 과목 | 파트 | 내용 |
|---|---|---|
| 물리학 | 역학 | 클램프 · 저울 · 도르래 · 역학 수레 · 단진자 · 받침대 · 회전축 · 막대자석 |
| 물리학 | 전기자기학 | 도선 · 나침반 · 회로 소자 10종(저항 R · 코일 L · 축전기 C · 직류전원 · 교류전원 · 전구 · 전류계 A · 전압계 V · 다이오드 · 미지소자) |
| 물리학 | 파동 및 광학 | 볼록렌즈 · 오목렌즈 · 볼록거울 · 오목거울 · 평면거울 · 물체(화살표) · 스크린 · 점광원 |
| 물리학 | 열역학 / 현대물리학 | "준비 중입니다." (아직 심볼 없음) |
| 화학 · 생명과학 · 지구과학 | 전 파트 | "준비 중입니다." (아직 심볼 없음) |

> 화학·생명과학·지구과학은 파트 틀만 마련된 상태이며 심볼은 이후 버전에서 채워집니다. 그때까지는 공통 도구와 퍼스널 오브젝트(6.4절)를 활용해 도해를 만들고 저장해 두는 방법을 권합니다.

#### 6.2.1 배치 방식 — 심볼마다 셋 중 하나

| 배치 방식 | 해당 심볼 | 조작 |
|---|---|---|
| **두 점 클릭** | 회로 소자 10종 | 첫 클릭 = 왼쪽 단자, 두 번째 클릭 = 오른쪽 단자. 두 점 사이에 소자가 그려집니다 |
| **드래그** | 렌즈·거울·물체·스크린·점광원, 받침대·회전축·막대자석, 도선·나침반·클램프·저울, 도르래·역학 수레, 단진자 | 누른 채 끌어 크기를 정하고 놓습니다 |
| **한 번 클릭** | 점(공통 도구의 '점' 버튼, N) | 클릭한 자리에 기본 크기 점 생성 |

| 단축키/모디파이어 | 동작 |
|---|---|
| `Ctrl` (회로 소자 두 점 클릭 중) | 15° 단위 각도 스냅 — 소자를 수평·수직·대각으로 반듯하게 |
| `Esc` | 진행 중인 클릭 배치(첫 점만 찍은 상태) 취소 |
| 드래그 없이 클릭만 | 아무것도 생성되지 않음 (드래그형 심볼) |
| `Space` + 드래그 | 배치 대신 화면 이동(팬) — 심볼이 그려지지 않습니다 |

**팁·주의사항**

- 심볼 하나를 배치하면 도구가 자동으로 선택(V)으로 돌아옵니다. **같은 심볼을 여러 개** 넣으려면 버튼을 다시 클릭하세요.
- 도르래·역학 수레는 드래그해도 **원본 비율이 자동 유지**됩니다(찌그러지지 않음). 나침반·클램프·저울도 비율이 고정됩니다.
- 단진자는 **드래그 시작점이 고정점(천장 지지대), 끝점이 추의 중심**입니다. 중앙 잔상·대칭 잔상·길이 라벨(L)은 기본으로 켜져 있고 오른쪽 인스펙터에서 끌 수 있습니다.
- 회로 팔레트에서 저항·코일·축전기·전압계·전류계 버튼은 그림 대신 **R·L·C·V·A 글자**로 표시됩니다.
- 배치 직후 `Ctrl+Z` 한 번이면 방금 만든 심볼이 통째로 사라집니다.
- 모바일 화면에서는 ☰ 도구 버튼으로 왼쪽 패널을 열고, 심볼을 고르면 패널이 자동으로 닫혀 바로 캔버스가 보입니다.

### 6.3 오브젝트 검색 (Ctrl+F) — 아코디언을 뒤지지 않고 바로 찾기

심볼이 어느 파트에 있는지 기억나지 않을 때, 이름이나 키워드로 검색해 곧바로 배치 모드로 들어가는 기능입니다. 퍼스널 오브젝트도 함께 검색됩니다.

**사용 절차**

1. `Ctrl+F`를 누르거나, 캔버스 하단 바의 **[오브젝트 검색]** 버튼을 클릭합니다.
2. 이름 또는 키워드를 입력합니다. (예: "렌즈", "저항", "ohm", "자석")
3. `↑`/`↓`로 결과를 고르고 `Enter`를 누르거나, 항목을 **더블클릭**합니다.
4. 배지가 **[즉시]**인 항목은 화면 중앙에 바로 생성되고, **[드래그]**인 항목은 해당 배치 도구가 활성화되며, **[퍼스널]**인 항목은 저장해 둔 묶음이 화면 중앙에 삽입됩니다.

| 단축키 | 동작 |
|---|---|
| `Ctrl+F` | 오브젝트 검색 열기 |
| `↑` / `↓` | 결과 목록 이동 |
| `Enter` | 선택한 항목 배치/삽입 |
| `Esc` | 검색 창 닫기 |
| `Ctrl+Shift+F` | (이 기능이 아님) 기출문항 검색 — 7장 참고 |

**팁·주의사항**

- 한국어·영어 키워드 모두 등록되어 있어 "coil"로도 "코일"로도 찾을 수 있습니다.
- 퍼스널 오브젝트는 **현재 과목에 저장된 것만** 검색 결과에 나타납니다.

### 6.4 퍼스널 오브젝트 — 내가 만든 도해를 저장하는 개인 저장소

여러 도형을 조합해 만든 도해(예: 빗면+수레+힘 화살표 세트)를 이름을 붙여 저장해 두고, 다음 작업에서 클릭 한 번으로 다시 불러오는 기능입니다. 저장 데이터는 이 브라우저 안(localStorage)에 보관됩니다.

#### 6.4.1 오브젝트 저장

1. 캔버스에서 저장할 오브젝트들을 선택합니다. (드래그로 여러 개를 함께 선택하거나 `Shift`+클릭으로 추가 선택)
2. 왼쪽 패널 **고급 기능 → [오브젝트 저장]**을 클릭합니다.
3. 모달에서 **이름**(최대 40자)을 입력하고 **분류**를 고릅니다. 처음이라면 기본 분류는 "기본"이며, "＋ 새 분류 만들기…"를 선택해 새 분류 이름(최대 20자, 예: "역학 세트")을 만들 수 있습니다.
4. **[저장]**을 누르면 현재 과목 정보와 함께 저장되고, 왼쪽 '퍼스널 오브젝트' 목록에 즉시 나타납니다.

| 조작 | 동작 |
|---|---|
| `Enter` (모달 안) | 저장 확정 |
| `Esc` / 바깥 클릭 | 저장 취소 |

**팁·주의사항**

- 아무것도 선택하지 않고 [오브젝트 저장]을 누르면 "저장할 오브젝트를 먼저 캔버스에서 선택하세요."라는 안내가 뜹니다.
- 그룹으로 묶인 오브젝트는 그룹 관계까지 그대로 저장·복원됩니다.
- 저장 항목은 **저장 시점의 과목**에 귀속됩니다. 물리학 모드에서 저장한 세트는 화학 모드에서는 보이지 않습니다(과목을 되돌리면 다시 보입니다).

#### 6.4.2 저장한 오브젝트 삽입

1. 왼쪽 패널에서 **퍼스널 오브젝트** 섹션을 펼칩니다.
2. 분류 아코디언을 열고 항목 이름 버튼을 클릭합니다.
3. 저장해 둔 묶음이 **현재 화면 중앙**에 삽입되고, 삽입된 오브젝트들이 선택된 상태가 됩니다. 그대로 드래그해 원하는 위치로 옮기면 됩니다.

| 조작 | 동작 |
|---|---|
| 항목 이름 클릭 | 화면 중앙에 삽입 (삽입 직후 선택 상태) |
| `Ctrl+Z` | 삽입 전체를 한 번에 취소 |
| ✕ 버튼 | 해당 항목 삭제 (확인 대화상자 표시) |

#### 6.4.3 오브젝트 저장소 — 썸네일로 미리 보며 관리

이름만으로는 어떤 도해였는지 기억나지 않을 때, 저장된 모든 항목을 **그림 미리보기(썸네일) 카드**로 훑어보고 삽입·삭제하는 관리 화면입니다.

1. 퍼스널 오브젝트 섹션 맨 위의 **[오브젝트 저장소]** 버튼을 클릭합니다.
2. 카드마다 썸네일, 이름, "분류 · 오브젝트 N개" 정보가 표시됩니다.
3. 상단의 **분류 드롭다운**과 **이름 검색창**으로 목록을 좁힐 수 있습니다.
4. 카드의 **[삽입]**을 누르면 화면 중앙에 삽입되며 모달이 닫히고, **[삭제]**를 누르면 확인 후 삭제됩니다.

| 조작 | 동작 |
|---|---|
| `Esc` / [닫기] / 바깥 클릭 | 저장소 닫기 |
| 분류 드롭다운 | 특정 분류만 표시 ("분류 전체"가 기본) |
| 이름 검색창 | 입력하는 즉시 실시간 필터 |

#### 6.4.4 백업과 복원 — 설정 파일로 안전하게

퍼스널 오브젝트는 브라우저 안에만 저장되므로, **브라우저 방문 기록/캐시를 삭제하면 함께 사라질 수 있습니다.** 정기적으로 파일로 백업해 두세요.

1. 상단 툴바에서 **설정 ▾ → 설정 저장하기**를 클릭합니다.
2. 항목 목록에서 **"퍼스널 오브젝트 라이브러리"**(필요하면 "과목 선택" 등도 함께)를 체크합니다.
3. **[저장]**을 눌러 JSON 설정 파일을 원하는 위치에 내려받습니다.
4. 복원할 때(또는 다른 컴퓨터에서)는 **설정 ▾ → 설정 불러오기**로 그 파일을 선택하면 됩니다. 불러오는 즉시 왼쪽 목록이 갱신됩니다.

**팁·주의사항**

- 설정 파일에는 체크한 항목만 담깁니다. 백업 목적이라면 "퍼스널 오브젝트 라이브러리"가 체크되어 있는지 꼭 확인하세요.
- 학교 컴퓨터와 집 컴퓨터를 오가며 쓸 때 이 설정 파일이 이동 수단입니다. 프로젝트 파일(파일 ▾ → 프로젝트 저장)과는 별개라는 점에 유의하세요 — 프로젝트 파일은 캔버스 내용을, 설정 파일은 라이브러리·환경을 담습니다.

## 7. 기출 라이브러리와 평가원 양식 — 검색·반입·미리보기·인쇄·전체 수정

이 장에서는 5E에 내장된 **기출 문항 라이브러리**에서 수능·모의평가 문항 그림을 찾아 캔버스로 가져오는 방법과, 완성한 그림을 **평가원 시험지 양식(수능 8절) 위에 실제 인쇄 크기로 미리보기** 하는 방법, 그리고 인쇄 전 마지막 손질에 쓰는 **전체 통일/수정** 기능을 다룹니다.

- **검색** — 과목·학년도·단원·개념 필터와 번호/해시태그 검색으로 기출 문항 찾기 (7.1~7.2)
- **반입** — 찾은 문항을 이미지로 삽입하거나 편집 가능한 오브젝트로 변환 (7.3~7.4)
- **미리보기** — 내 그림을 시험지 지면에 실제 크기로 얹어 확인 (7.5)
- **인쇄** — 인쇄 비교 배경 등록과 인쇄용 내보내기 흐름 (7.6~7.7)
- **전체 수정** — 여러 오브젝트의 굵기·색·글씨·각도를 한 번에 (7.8)

### 7.1 기출 문항 검색 창 열기

**무엇인가:** 물리학·화학·생명과학·지구과학 I/II 8개 과목, 2007~2027학년도 6월 모평·9월 모평·수능 기출 문항 이미지 약 4,000개가 들어 있는 내장 라이브러리를 여는 기능입니다.

**사용 절차**

1. 캔버스 **아래쪽 바**에서 **[기출문항 검색]** 버튼을 클릭합니다. (돋보기 모양의 [오브젝트 검색] 버튼 바로 옆, 문서 모양 아이콘)
2. 또는 키보드에서 **Ctrl+Shift+F**를 누릅니다.
3. 처음 열 때는 "문항 목록 불러오는 중…"이 잠시 표시된 뒤 검색 화면이 준비됩니다.
4. 상단의 **과목 선택** 드롭다운이 자동으로 현재 과목의 첫 과목(예: 물리학 모드면 "물리1")으로 맞춰지고, 해당 과목 문항이 카드 형태로 나타납니다.
5. 다 봤으면 **[닫기]** 버튼, 창 바깥 클릭, 또는 **Esc**로 닫습니다.

**단축키**

| 키 | 동작 |
|---|---|
| Ctrl+Shift+F | 기출문항 검색 창 열기 |
| Ctrl+F | (참고) 캔버스 위 오브젝트 검색 — 기출 검색과 짝을 이루는 단축키 |
| Esc | 검색 창 닫기 |

**팁·주의사항**

- 과목 드롭다운에는 **왼쪽 패널 맨 위에서 고른 과목 모드**(물리학/화학/생명과학/지구과학)에 해당하는 과목만 나타납니다. 예를 들어 물리학 모드에서는 "물리1·물리2"만 보입니다. 다른 과목의 기출을 찾으려면 먼저 왼쪽 패널에서 과목을 바꾸세요.
- **과목을 선택하기 전에는 목록이 표시되지 않습니다.** "과목을 먼저 선택하세요."라는 안내가 뜨면 과목 드롭다운부터 고르면 됩니다.
- 검색 결과가 많으면 **앞 60개만** 카드로 표시되고, "검색 결과 N개 — 앞 60개만 표시"라는 안내가 나옵니다. 필터나 검색어로 좁혀 주세요.
- "라이브러리를 찾을 수 없습니다"라는 오류가 뜨면 라이브러리 파일이 없는 환경(로컬 복사본 등)입니다. 공개 배포 사이트에서는 정상 동작합니다.

### 7.2 검색 — 필터·번호 검색·해시태그 검색

**무엇인가:** 드롭다운 필터 4종(과목·학년도 범위·단원·개념)과 검색창(번호 코드 또는 키워드)을 **모두 겹쳐서(AND)** 문항을 좁혀 가는 검색 방식입니다.

**사용 절차 — 필터로 좁히기**

1. **과목 선택**에서 과목(예: 물리1)을 고릅니다.
2. **년도(시작) ~ 년도(끝)**에서 학년도 범위를 고릅니다. 한쪽만 골라도 되고, 순서가 뒤집혀 있어도 자동으로 바로잡습니다.
3. **단원 전체** 드롭다운에서 단원(예: 역학, 유전, 고체지구 등)을 고릅니다.
4. **개념 전체** 드롭다운에서 세부 개념(예: 도르래, 마찰력, DNA복제 등)을 고릅니다. 단원을 먼저 고르면 그 단원의 개념만 목록에 남습니다.
5. 조건을 모두 지우려면 **[필터 초기화]**를 누릅니다.

**사용 절차 — 번호(압축 코드)로 바로 찾기**

1. 검색창에 숫자만 입력합니다. 자릿수에 따라 자동으로 해석됩니다.
2. 결과 카드가 바로 갱신됩니다. (문항 제목은 "2027학년도 6월 모평 생명1 1번" 형식입니다.)

| 입력 형식 | 예시 | 해석 |
|---|---|---|
| 4자리 (YYMM) | `2611` | 2026학년도 11월(수능) 전 문항 |
| 6자리 (YYYYMM) | `202611` | 2026학년도 11월 전 문항 |
| 6자리 (YYMMNN) | `261101` | 2026학년도 수능 1번 |
| 8자리 (YYYYMMNN) | `20261101` | 2026학년도 수능 1번 |

- 시행 월은 **6월(모평)·9월(모평)·11월(수능)** 세 가지입니다.
- 숫자 사이에 공백, 점(`.`), 하이픈(`-`)을 넣어도 됩니다. (예: `26-11-01`)
- 글자가 섞이면 압축 코드가 아니라 아래의 키워드 검색으로 처리됩니다.

**사용 절차 — 키워드·해시태그로 찾기**

1. 검색창에 단어를 입력합니다. 문항 번호(id)·제목·태그에서 찾습니다.
2. 여러 단어를 **공백** 또는 **`#`**로 이어 쓰면 전부 만족하는 문항만 남습니다(AND). 예: `#역학#도르레#마찰력`
3. "물리11번"처럼 붙여 써도 매치되도록 띄어쓰기를 무시한 비교도 함께 합니다.

**단축키**

| 키/조작 | 동작 |
|---|---|
| (입력 즉시) | 검색창은 타이핑할 때마다 실시간 갱신 — 별도 Enter 불필요 |
| Esc | 검색 창 닫기 |

**팁·주의사항**

- 필터와 검색어는 **모두 동시에 적용**됩니다. 결과가 0개면 "필터 초기화" 후 다시 시도해 보세요.
- 개념 태그는 과목별 단원 체계(물리: 역학·전자기학·광학·현대물리학 / 화학·생명·지구 각 4개 단원)에 따라 정리되어 있습니다.

### 7.3 반입 ① — 이미지 삽입

**무엇인가:** 선택한 기출 문항을 **이미지 오브젝트**로 캔버스에 넣는 기능입니다. 그림을 참고 자료로 옆에 두거나, 문항 지문 전체를 그대로 쓰고 싶을 때 사용합니다.

**사용 절차**

1. 검색 결과에서 원하는 **카드를 클릭**해 선택합니다. (선택된 카드는 테두리가 강조됩니다.)
2. 상단의 **[이미지 삽입]** 버튼을 누릅니다. (카드를 선택해야 버튼이 활성화됩니다.)
3. "…불러오는 중…" 표시 후 검색 창이 닫히고, 이미지가 캔버스에 놓입니다.
4. 위치·크기는 보통 이미지처럼 선택 도구로 조절하면 됩니다.

**단축키**

| 키/조작 | 동작 |
|---|---|
| 카드 클릭 | 문항 선택 (버튼 활성화) |
| Ctrl+Z | 삽입 취소 (삽입 1회 = 실행 취소 1단계) |

**팁·주의사항**

- 삽입 위치는 **마지막 마우스 위치**(없으면 화면 중앙)이고, 크기는 아트보드에 맞게 자동 축소됩니다. 가로세로 비율은 잠긴 상태로 들어옵니다.
- 이미지는 파일 경로 참조가 아니라 **데이터 자체가 프로젝트에 포함**됩니다. 따라서 프로젝트 파일(.json)을 다른 컴퓨터로 옮겨도 그림이 깨지지 않습니다. 대신 이미지를 많이 넣으면 프로젝트 파일이 커집니다.

### 7.4 반입 ② — 오브젝트 변환

**무엇인가:** 선택한 기출 문항 이미지를 **이미지 객체화** 기능으로 넘겨, 그림 속 선·글자를 **편집 가능한 개별 오브젝트**(선·텍스트·그룹)로 분해해 넣는 기능입니다. 기출 도해를 변형해서 새 문항을 만들 때 씁니다.

**사용 절차**

1. 검색 결과에서 카드를 클릭해 선택합니다.
2. 상단의 **[오브젝트 변환]** 버튼을 누릅니다.
3. 검색 창이 닫히고 **이미지 객체화** 모달이 해당 문항 이미지가 미리 담긴 상태로 열립니다.
4. 이후 분석·삽입 과정은 일반 이미지 객체화와 동일합니다. (이미지 객체화의 자세한 사용법은 해당 장을 참고하세요.)

**단축키**

| 키/조작 | 동작 |
|---|---|
| 카드 클릭 | 문항 선택 (버튼 활성화) |
| Ctrl+T | (참고) 이미지 객체화 모달을 직접 여는 단축키 |
| Shift+G | (참고) 변환 결과 그룹 해제 |

**팁·주의사항**

- 변환 결과물은 하나의 **그룹**으로 묶여 들어오므로, 부분 편집 전에 그룹을 해제하면 편합니다.
- 글자가 많은 문항은 변환 후 정리할 것이 많아집니다. 그림(도해) 위주의 문항에서 효과가 좋습니다.

### 7.5 시험지 미리보기 — 평가원 양식 위에 실제 크기로 확인

**무엇인가:** 이미지로 내보내기 전에, 내 그림을 실제 시험지 지면(수능 8절, 272×394 mm) 위에 **실제 인쇄 크기 그대로** 얹어 보는 기능입니다. 시험지 글자 크기는 정해져 있으므로, 그림이 본문과 어울리는 크기인지 인쇄 전에 확인할 수 있습니다.

**사용 절차**

1. 상단 **파일 → 이미지로 내보내기**를 클릭하거나 **Alt+P**를 누릅니다.
2. 내보내기 대화상자에서 해상도(dpi)·참고 이미지 포함 여부를 정한 뒤 **[미리보기]** 버튼을 누릅니다.
3. 화면이 어두워지면 "미리볼 영역을 드래그하십시오" 안내에 따라 **미리볼 영역을 드래그**로 그립니다.
4. 드래그를 놓으면 조정 단계가 됩니다. 8개 핸들로 크기를 다듬고, 사각형 몸체를 끌어 위치를 옮기고, 위쪽 파란 라벨의 숫자를 클릭해 **가로·세로 mm를 직접 입력**할 수도 있습니다.
5. **Enter**로 확정하면 "시험지 미리보기" 창이 열리고, 지정한 영역이 시험지 위에 실제 크기로 얹혀 나타납니다.
6. **배경 양식** 드롭다운에서 시험지 종류(내장 양식 + 직접 등록한 배경, 7.6 참조)를 고릅니다.
7. 그림을 **드래그해 원하는 위치로** 옮겨 보며 본문·여백과의 균형을 확인합니다. **크기는 실제 인쇄 크기로 고정**되어 있어 바꿀 수 없습니다.
8. **[위치 초기화]**로 기본 위치로 되돌리고, 확인이 끝나면 **[닫기]** 또는 **Esc**로 닫습니다.

**단축키/조작**

| 단계 | 키/조작 | 동작 |
|---|---|---|
| 영역 지정 | 드래그 | 영역 그리기 / 다시 그리기 |
| 영역 지정 | 핸들 드래그 | 크기 조절 (8방향) |
| 영역 지정 | 몸체 드래그 | 영역 이동 |
| 영역 지정 | 치수 라벨 클릭 → 숫자 입력, Tab | 가로 mm 입력 → 세로 mm 입력 |
| 영역 지정 | Enter | 영역 확정 → 미리보기 열기 |
| 영역 지정 | Esc / 우클릭 | 취소 (내보내기 대화상자로 복귀) |
| 미리보기 창 | 그림 드래그 | 위치 이동 (크기 변경 불가) |
| 미리보기 창 | Esc | 미리보기만 닫기 |

**팁·주의사항**

- 미리보기 화면의 그림은 내보내기와 **완전히 같은 변환 과정**으로 만들어집니다. 즉 "미리 본 그대로" 내보내집니다. 글꼴 대체나 잘림 문제도 미리보기에서 그대로 드러납니다.
- 미리보기는 순수 확인용입니다. 캔버스·실행 취소·프로젝트 저장에 아무 흔적도 남기지 않습니다.
- 그림이 시험지에 비해 너무 크거나 작다면, 미리보기를 닫고 **캔버스에서 오브젝트 크기를 조정**한 뒤 다시 미리보기 하세요.
- 그림 아래 안내줄에 **그림 실제 크기(mm)와 페이지 크기(mm)**가 표시되므로 수치로도 비교할 수 있습니다.
- 내장 시험지 배경("물리학 I · 과학탐구 (수능 8절)")은 저작권 문제로 **저장소에 포함되지 않는 로컬 자산**입니다. "배경 이미지를 불러오지 못했습니다"가 뜨는 환경이라면, 7.6의 **인쇄 비교 배경 이미지 등록**으로 자신의 시험지 스캔본을 등록해 쓰면 됩니다.

### 7.6 인쇄 비교 배경 이미지 등록

**무엇인가:** 실제로 인쇄해 본 시험지(스캔본·사진)를 배경 후보로 등록해 두면, 시험지 미리보기(7.5)에서 배경으로 골라 **내 학교 양식 그대로** 크기를 비교할 수 있는 기능입니다.

**사용 절차**

1. 상단 **설정 → 기본값 설정**을 엽니다.
2. 아래쪽 **"인쇄 비교 이미지"** 항목으로 이동합니다.
3. **파일 선택**에서 이미지 파일(PNG/JPEG/WebP)을 고릅니다.
4. **이름(선택)**과 그 종이의 **실제 크기 가로 mm / 세로 mm**를 입력합니다. 비워 두면 A4(210×297 mm)로 저장됩니다.
5. **[추가]**를 누르면 목록에 등록됩니다.
6. 필요 없어진 배경은 목록 오른쪽 **✕** 버튼 → 확인 창에서 "예"로 삭제합니다.
7. 이후 **이미지로 내보내기 → 미리보기**의 "배경 양식" 드롭다운에 등록한 배경이 나타납니다.

**단축키**

| 키/조작 | 동작 |
|---|---|
| (전용 단축키 없음) | 설정 메뉴에서 진입 |
| Esc | 기본값 설정 창 닫기 |

**팁·주의사항**

- **가로·세로 mm는 반드시 실측값**을 넣으세요. 미리보기는 이 값을 기준으로 그림을 실제 배율로 얹기 때문에, 크기가 틀리면 비교 자체가 어긋납니다. (시험지 8절은 272×394 mm, A4는 210×297 mm)
- 등록 이미지는 긴 변 2000px 이하로 자동 축소되어 브라우저 저장소(localStorage)에 보관됩니다. "저장 공간이 부족합니다"가 뜨면 기존 배경을 삭제한 뒤 다시 추가하세요.
- 등록한 배경은 **설정 → 설정 저장하기/설정 불러오기**의 백업 대상에 포함됩니다. 브라우저 캐시를 지워도 설정 파일로 복원할 수 있습니다.

### 7.7 인쇄로 가는 길 — 내보내기·복사

**무엇인가:** 5E에는 브라우저 인쇄 버튼이 따로 없습니다. **미리보기로 크기를 확정 → PNG로 내보내기(또는 복사) → 한글(HWP)·PPT에 붙여 넣고 문서에서 인쇄**하는 흐름이 표준입니다.

**사용 절차**

1. **Alt+P**로 이미지로 내보내기 대화상자를 엽니다.
2. **형식**을 고릅니다. 한글·PPT 삽입과 인쇄에는 **PNG**(흰 배경, 인쇄 표준), 일러스트 편집용에는 SVG.
3. PNG라면 **해상도**를 고릅니다: 200 / **300(기본)** / 400 dpi. 인쇄물은 300 dpi면 충분합니다.
4. **배경/참고 이미지 포함** 체크로 참고용 이미지를 결과물에 넣을지 정합니다.
5. 다음 중 하나로 마무리합니다.
   - **[내보내기]** — 아트보드 전체를 파일로 저장
   - **[영역 지정]** — 드래그로 지정한 영역만 저장 (조작법은 7.5의 영역 지정과 동일: 핸들·mm 입력·Enter 확정)
   - **[복사]** — PNG를 클립보드에 복사 → 한글·PPT에서 **Ctrl+V**로 바로 붙여넣기
6. 한글 문서에 삽입한 뒤 문서 인쇄 기능으로 출력합니다.

**단축키**

| 키/조작 | 동작 |
|---|---|
| Alt+P | 이미지로 내보내기 대화상자 열기 |
| Ctrl+V (한글/PPT에서) | [복사]한 PNG 붙여넣기 |
| Enter / Esc / 우클릭 | 영역 지정 확정 / 취소 / 취소 |
| Ctrl+S | (참고) 프로젝트 저장 — 이미지 내보내기와 별개 |

**팁·주의사항**

- **인쇄 전 반드시 7.5 미리보기로 실제 크기를 확인**하는 습관을 들이면, 인쇄 후 "그림이 본문보다 크다/작다"를 다시 뽑는 일이 없습니다. 미리보기와 내보내기는 같은 dpi·옵션 설정을 공유합니다.
- [복사]가 안 되는 브라우저 환경에서는 안내 창이 뜹니다. 이때는 [내보내기]로 파일 저장 후 삽입하세요.
- 파일 이름은 열 때마다 현재 날짜·시각(YYYYMMDD_HHmm)으로 자동 제안되며, 원하는 이름으로 바꿔도 됩니다.

### 7.8 전체 통일/수정 — 여러 오브젝트를 한 번에 고치기

**무엇인가:** 선택한 오브젝트(선택이 없으면 캔버스 전체)의 선 굵기·색 어둡기·글씨 크기·각도 등을 **한 번에 같은 값으로 통일**하거나 **일정량씩 증감**하는 기능입니다. 기출을 반입해 편집하다 보면 원본과 새로 그린 부분의 굵기·톤이 제각각이 되는데, 인쇄 직전에 이것으로 한 번에 맞춥니다.

**사용 절차**

1. (선택) 대상 오브젝트들을 먼저 선택해 둡니다. 아무것도 선택하지 않으면 캔버스 전체가 대상이 됩니다.
2. 왼쪽 패널 **고급 기능 → [전체 통일/수정]** 버튼을 누릅니다.
3. 창 상단에서 모드를 고릅니다.
   - **전체 통일** — 모든 오브젝트를 같은 수치로 통일
   - **전체 수정** — 모든 오브젝트의 수치를 입력한 만큼 ± 증감
4. "대상: 선택한 오브젝트 N개" 또는 "대상: 캔버스 전체 N개" 문구로 적용 범위를 확인합니다.
5. 바꿀 항목의 **체크박스를 켜고 값을 입력**합니다. (값을 입력하면 체크가 자동으로 켜집니다.)
6. **[적용]**을 누릅니다. 잘못 적용했으면 **Ctrl+Z** 한 번으로 전부 되돌아갑니다.

**적용 가능 항목**

| 항목 | 단위 | 전체 통일 | 전체 수정(±) | 비고 |
|---|---|---|---|---|
| 선 굵기 | mm | O | O | 0 미만으로는 내려가지 않음 |
| 선 색(어둡기) | 0~255 | O | O | 값이 클수록 진함(255=검정) |
| 면 색(어둡기) | 0~255 | O | O | 값이 클수록 진함 |
| 글씨 크기 | pt | O | O | 텍스트와 라벨 글씨에 적용, 최소 크기 하한 있음 |
| 각도 | ° | O (절대각) | O (상대 회전) | 직선류는 중점 기준으로 양 끝점 회전 |
| 글씨체 | — | O | — | 텍스트·수식 오브젝트에 적용 |
| 위치 고정 | 켜기/끄기 | O | — | |
| 오브젝트 잠금 | 켜기/끄기 | O | — | |

**단축키**

| 키/조작 | 동작 |
|---|---|
| (전용 단축키 없음) | 고급 기능 패널에서 진입 |
| Esc | 창 닫기 (적용 안 함) |
| Ctrl+Z | 적용 결과 한 번에 되돌리기 (적용 1회 = 실행 취소 1단계) |

**팁·주의사항**

- **잠긴 오브젝트는 보호됩니다.** 잠긴 오브젝트에는 "위치 고정"과 "오브젝트 잠금" 항목만 적용되므로, 이 기능으로 **전체 잠금 해제**도 할 수 있습니다(통일 모드에서 "오브젝트 잠금 → 끄기"). 대상 문구에 "(잠긴 N개는 잠금 항목만 변경)"으로 표시됩니다.
- 전체 수정(±) 모드에서 값이 0인 항목은 체크되어 있어도 건너뜁니다.
- 각도 통일 시 직선(전기회로 선 포함)은 지정한 절대각으로 눕고, 도형·자·각도기 등은 회전값이 그 각도로 맞춰집니다.
- 기출 반입 후 마무리 순서로는 **① 전체 통일로 선 굵기·어둡기 정리 → ② 7.5 미리보기로 크기 확인 → ③ 7.7 내보내기·인쇄**가 가장 손이 덜 갑니다.

## 8. 저장·내보내기·설정 — 프로젝트 파일·PNG/SVG·설정 IO·잠금/보호·스타일

이 장에서는 작업물을 **다시 편집할 수 있는 프로젝트 파일**로 보관하는 방법, 한글(HWP)·PPT에 넣을 **완성 이미지(PNG/SVG)로 내보내는** 방법, 그리고 개인 설정을 백업·복원하고 오브젝트를 실수로부터 보호하는 방법을 다룹니다.

> **먼저 구분해 둘 것 — "프로젝트 저장"과 "이미지로 내보내기"는 다릅니다.**
> - **프로젝트 저장(.json)** : 도면의 설계도. 나중에 5E에서 다시 열어 이어서 편집할 수 있습니다.
> - **이미지로 내보내기(.png/.svg)** : 완성된 그림 파일. 문서에 붙여 넣는 용도이며, 5E에서 다시 편집할 수 없습니다.
> 중요한 작업은 **둘 다** 저장해 두는 것이 안전합니다.

### 8.1 프로젝트 저장

지금 그리고 있는 도면 전체를 **다시 편집 가능한 JSON 파일**로 컴퓨터에 내려받습니다.

**사용 절차**

1. 화면 위쪽 도구줄에서 **[파일 ▾]** 버튼을 누릅니다.
2. **[프로젝트 저장]**을 클릭합니다. (또는 언제든 `Ctrl+S`)
3. 브라우저의 다운로드 폴더에 `physics_drawing.json` 파일이 저장됩니다.

**단축키**

| 키 | 동작 |
|---|---|
| `Ctrl+S` | 프로젝트 저장 (브라우저의 "페이지 저장" 대화상자 대신 5E 저장이 실행됨) |

**팁·주의사항**

- 저장되는 것: 모든 오브젝트, 안내선, 레이어, 아트보드(용지) 크기.
- 저장되지 **않는** 것: 되돌리기(Undo) 기록, 현재 선택 상태, 화면 확대·이동 위치. 파일을 다시 열면 이 값들은 초기화됩니다.
- 파일 이름이 항상 `physics_drawing.json`으로 같으므로, 여러 도면을 관리하려면 다운로드 후 **파일 이름을 직접 바꿔** 두세요(예: `2단원_빗면.json`).
- 5E는 웹앱이라 브라우저를 닫으면 작업 내용이 사라집니다. 수업 자료처럼 중요한 도면은 **작업 중간중간 `Ctrl+S`** 를 습관화하세요.

### 8.2 프로젝트 불러오기

저장해 둔 프로젝트 JSON 파일을 열어 **이어서 편집**합니다.

**사용 절차**

1. **[파일 ▾] → [프로젝트 불러오기]**를 클릭합니다.
2. 파일 선택 창에서 저장해 둔 `.json` 파일을 고릅니다.
3. 도면이 캔버스에 그대로 복원됩니다.

**팁·주의사항**

- **현재 작업이 통째로 교체됩니다.** 불러오기 전에 지금 도면을 먼저 저장하세요. 별도의 경고 창은 뜨지 않습니다.
- 예전 버전 5E로 저장한 파일도 열 때 자동으로 최신 형식으로 변환됩니다(원본 파일은 바뀌지 않음).
- 잘못된 파일(프로젝트 파일이 아닌 JSON, 손상된 파일)을 고르면 오류 안내만 뜨고 **현재 도면은 그대로 유지**됩니다.
- 파일을 열면 되돌리기 기록과 선택 상태가 초기화되고, 화면 위치·배율은 현재 상태 그대로 유지됩니다.

### 8.3 이미지로 내보내기 (PNG / SVG)

완성한 그림을 **문서에 넣을 이미지 파일**로 저장합니다. 아트보드(용지) 영역이 실제 물리 크기(mm) 그대로 내보내져, 한글(HWP)·PPT에 붙였을 때 **실제 인쇄 크기와 일치**합니다.

**사용 절차**

1. **[파일 ▾] → [이미지로 내보내기]**를 클릭합니다. (또는 `Alt+P`)
2. 대화상자에서 옵션을 정합니다.
   - **파일 이름** — 기본값은 현재 날짜·시각(예: `20260710_1030`)이며 자유롭게 고칠 수 있습니다.
   - **형식** — `PNG`(문서 삽입용, 흰 배경) 또는 `SVG`(확대해도 깨지지 않는 벡터, 배경 투명).
   - **해상도** — PNG일 때만: 200 / 300 / 400 dpi (기본 300 dpi). SVG는 벡터라 해상도 선택이 사라집니다.
   - **배경/참고 이미지 포함** — 체크를 끄면 트레이싱용으로 깔아 둔 참고(배경) 이미지가 결과물에서 빠집니다. 기본은 포함.
3. **[내보내기]**를 누릅니다.
4. 크롬·엣지에서는 저장 위치와 파일명을 고르는 창이 뜨고, 그 외 브라우저에서는 다운로드 폴더로 바로 내려받습니다.

**단축키**

| 키 | 동작 |
|---|---|
| `Alt+P` | 이미지로 내보내기 대화상자 열기 (텍스트 입력 중에는 동작하지 않음) |
| `Esc` | 대화상자 닫기 |

**팁·주의사항**

- 내보내지는 것은 **완성된 도면뿐**입니다. 격자, 안내선, 선택 핸들 같은 편집 화면 요소는 절대 포함되지 않습니다.
- **숨긴 레이어**의 오브젝트와, 속성에서 "내보내기 제외"로 설정한 오브젝트는 결과물에서 빠집니다.
- 아트보드 밖으로 삐져나간 부분은 잘려 나갑니다.
- 시험 문항용이라면 **PNG 300 dpi**가 무난합니다. 인쇄가 흐리면 400 dpi로 올리세요.
- 글자는 폰트가 파일에 심어지지 않고 이름만 실려 가므로, SVG를 다른 컴퓨터의 일러스트 프로그램에서 열면 글꼴이 대체될 수 있습니다. 문서 삽입이 목적이면 PNG가 안전합니다.

#### 8.3.1 영역 지정 내보내기

아트보드 전체가 아니라 **드래그로 지정한 사각형 영역만** 잘라 내보냅니다.

**사용 절차**

1. 내보내기 대화상자에서 **[영역 지정]**을 누릅니다. 화면이 어두워집니다.
2. 저장할 영역을 마우스로 **드래그**해 사각형을 그립니다.
3. 손을 떼면 조정 단계가 됩니다 — 8개의 흰 핸들로 크기를 조절하거나, 사각형 안쪽을 끌어 위치를 옮길 수 있습니다.
4. 사각형 위에 실제 크기(mm)가 실시간 표시됩니다. 숫자를 클릭하면 **가로 mm를 직접 입력**할 수 있고, `Tab`으로 세로 입력으로 넘어갑니다.
5. `Enter`를 누르면 그 영역이 대화상자의 설정(형식·해상도)대로 내보내집니다.

**단축키·조작**

| 키/조작 | 동작 |
|---|---|
| 드래그 | 영역 새로 그리기 / (조정 단계에서) 이동·크기 조절 |
| 숫자 클릭 후 입력, `Tab` | 가로·세로 크기를 mm 단위로 정확히 입력 |
| `Enter` | 확정하고 내보내기 |
| `Esc` 또는 마우스 우클릭 | 취소하고 대화상자로 복귀 |

#### 8.3.2 복사 (클립보드로 바로 붙여넣기)

파일을 만들지 않고 **PNG를 클립보드에 복사**해, 한글(HWP)·PPT에 `Ctrl+V`로 바로 붙여 넣습니다.

**사용 절차**

1. 내보내기 대화상자에서 해상도·참고 이미지 옵션을 정합니다.
2. **[복사]**를 누릅니다. 버튼이 "복사됨!"으로 잠시 바뀝니다.
3. 한글이나 PPT로 가서 `Ctrl+V`로 붙여 넣습니다.

**팁·주의사항**

- 클립보드 규격상 **PNG만** 복사됩니다(SVG 불가).
- 브라우저 환경에 따라 클립보드 복사가 지원되지 않을 수 있습니다. 이때는 안내가 뜨며, [내보내기]로 파일 저장을 이용하면 됩니다.

#### 8.3.3 미리보기 (실제 시험지 위에 얹어 보기)

내보내기 전에, 그림을 **실제 시험지(수능 8절, 272×394mm) 위에 실제 인쇄 크기로** 얹어 본문 글자·여백과 어울리는지 확인합니다.

**사용 절차**

1. 내보내기 대화상자에서 **[미리보기]**를 누릅니다.
2. 미리 볼 영역을 드래그로 지정하고 `Enter`로 확정합니다(조작법은 영역 지정과 동일).
3. 시험지 배경 위에 그림이 실제 배율로 표시됩니다. 그림을 드래그해 **위치만** 옮겨 볼 수 있고, 크기는 실제 인쇄 크기로 고정됩니다.
4. 위쪽 **배경 양식** 목록에서 다른 시험지 양식을 골라 볼 수 있습니다.
5. **[닫기]** 또는 `Esc`로 닫습니다. **[위치 초기화]**를 누르면 그림이 처음 자리로 돌아갑니다.

**팁·주의사항**

- 미리보기는 내보내기와 **완전히 같은 변환 경로**를 쓰므로, 화면에서 본 모습 그대로 PNG가 만들어집니다.
- 미리보기는 순수 확인용입니다 — 도면, 되돌리기 기록, 저장 파일 어디에도 흔적을 남기지 않습니다.
- 직접 인쇄해 본 시험지 사진을 배경으로 등록할 수도 있습니다 → 8.5의 "인쇄 비교 이미지" 참고.

### 8.4 설정 저장하기 / 불러오기 (설정 백업)

선 굵기 기본값·테마·퍼스널 오브젝트 같은 **개인 설정을 파일로 백업하고 복원**합니다.

5E의 개인 설정은 브라우저 안(localStorage)에 저장되기 때문에, **브라우저 캐시·사이트 데이터를 지우면 함께 사라집니다.** 공들여 만든 퍼스널 오브젝트가 있다면 반드시 파일로 백업해 두세요.

**저장 절차**

1. **[설정 ▾] → [설정 저장하기]**를 클릭합니다.
2. 저장할 항목을 체크합니다. (저장할 내용이 없는 항목은 회색으로 비활성화됩니다)
   - 기본값 설정 (선 굵기·글꼴 등)
   - 화면 테마 (다크/화이트)
   - 과목 선택
   - 퍼스널 오브젝트 라이브러리
   - 인쇄 비교 배경 이미지
3. **[저장]**을 누르고 저장 위치를 지정합니다. `5E-settings-YYYYMMDD.json` 형식의 파일이 만들어집니다.

**복원 절차**

1. **[설정 ▾] → [설정 불러오기]**를 클릭합니다.
2. 백업해 둔 설정 파일을 선택합니다.
3. "설정을 불러왔습니다" 안내가 뜨면 완료입니다.

**팁·주의사항**

- 5E 설정 파일이 아닌 JSON(예: 프로젝트 파일)을 고르면 안내만 뜨고 **아무것도 바뀌지 않습니다.**
- 화면 테마는 불러오는 즉시 적용됩니다. 기본값 설정은 다음에 "기본값 설정" 창을 열 때 복원된 값으로 표시됩니다.
- 다른 컴퓨터로 옮길 때도 이 기능을 쓰면 됩니다: 기존 PC에서 저장 → 새 PC의 5E에서 불러오기.

### 8.5 기본값 설정

새 오브젝트를 만들 때 쓸 **기본 스타일 값**(선 굵기·글꼴·격자 등)을 정해 두는 곳입니다.

**사용 절차**

1. **[설정 ▾] → [기본값 설정]**을 클릭합니다.
2. 항목을 수정합니다. 오른쪽 **미리보기**(빗면 위 물체 예시 그림)에 변경 결과가 즉시 반영되어 감을 잡을 수 있습니다.
   - 기본 선 굵기 (mm, 0.1~0.5)
   - 기본 선 명도 / 기본 채우기 명도 (0~255)
   - 기본 글자 크기 (mm) / 기본 글씨체 / 기본 글자 스타일(굵기)
   - 앱 시작 시 격자 표시 / 격자 진하기 (1~10) / 격자 간격 (mm)
   - 자 눈금 간격 (mm) / 각도기 눈금 간격 (°)
3. **[저장]**을 누르면 브라우저에 보관됩니다. [취소]나 `Esc`는 아무것도 바꾸지 않습니다.

**팁·주의사항**

- 창 상단에 안내되어 있듯, 현재 버전에서는 이 값들이 **저장만 되고 아직 새 도형 생성·격자 표시에는 반영되지 않습니다**(다음 업데이트에서 적용 예정). 미리 취향대로 맞춰 두면 적용 업데이트 때 그대로 쓰입니다.
- 이 설정은 8.4의 "설정 저장하기"로 백업할 수 있습니다.

**인쇄 비교 이미지 등록** — 같은 창 아래쪽에서, 실제로 인쇄해 본 시험지를 스캔·촬영한 이미지를 등록할 수 있습니다.

1. 이미지 파일(PNG/JPG/WebP)을 선택합니다.
2. 이름(선택)과 **실제 종이의 가로·세로 mm**를 입력합니다. (배율 비교의 기준이 되므로 실측값을 넣어야 합니다)
3. **[추가]**를 누릅니다. 등록한 이미지는 "이미지로 내보내기 → 미리보기"의 배경 양식 목록에 나타납니다.
4. 목록의 ✕ 버튼으로 삭제할 수 있습니다(확인 창이 뜹니다).

### 8.6 오브젝트 잠금 / 보호

완성해 둔 오브젝트를 **실수로 움직이거나 고치지 못하게** 막습니다. 보호는 두 단계입니다.

| 보호 | 강도 | 효과 |
|---|---|---|
| **위치 고정** | 약함 | 크기 조절·회전 등 변형은 가능하지만 **위치가 제자리에 고정**됩니다(변형해도 중심이 움직이지 않음) |
| **오브젝트 잠금** | 강함 | 변형·이동이 **모두 금지**됩니다. 선택은 가능하므로 언제든 잠금을 해제할 수 있습니다 |

**사용 절차**

1. 선택 도구(`V`)로 보호할 오브젝트를 선택합니다.
2. 오른쪽 속성 패널의 **[오브젝트 보호]** 섹션을 엽니다.
3. **위치 고정** 또는 **오브젝트 잠금** 체크박스를 켭니다. (여러 개를 선택했으면 한꺼번에 적용)
4. 해제하려면 다시 선택해 체크를 끕니다.

**단축키**

| 키 | 동작 |
|---|---|
| `K` | 선택한 오브젝트의 잠금을 켜기/끄기 (선택 도구 `V`에서만) |

**팁·주의사항**

- 잠금·해제도 되돌리기(`Ctrl+Z`)가 됩니다.
- 캔버스의 오브젝트 전체를 한 번에 잠그거나 풀려면 8.7의 **전체 통일/수정**에서 "오브젝트 잠금" 항목을 쓰세요. 잠긴 오브젝트도 잠금 항목만은 바꿀 수 있어서, **일괄 잠금 해제**가 가능합니다.
- **이미지 전용 잠금**: 속성 패널의 이미지 섹션에도 별도의 "잠금" 체크가 있습니다. 트레이싱용 배경 이미지를 잠그면 캔버스에서 클릭·드래그로 선택되지 않게 되어(그림만 보임), 그 위에 마음 놓고 그릴 수 있습니다. 해제는 속성 패널의 이미지 목록에서 합니다.
- 이런 참고(배경) 이미지는 내보내기 대화상자의 **"배경/참고 이미지 포함"** 체크를 끄는 것으로 완성 이미지에서 한 번에 제외할 수 있습니다.

### 8.7 스타일 일괄 변경 — 전체 통일/수정

여러 오브젝트의 선 굵기·색·글씨 크기·각도·잠금을 한 번에 통일하거나 증감하는 기능입니다. 자세한 사용법은 **7.8절 「전체 통일/수정」**을 참고하세요 — 기출 문항을 반입해 편집한 뒤 인쇄 전 마무리 단계에서 가장 많이 쓰입니다.

### 8.8 화면 테마 (흑백 모드)

편집 화면의 밝기 테마를 **다크(기본)와 화이트** 사이에서 전환합니다. 완성 이미지에는 영향이 없고, 화면 표시만 바뀝니다.

**사용 절차**

1. 화면 위쪽 도구줄 오른편의 **원형 아이콘(흑백 모드) 버튼**을 클릭합니다.
2. 누를 때마다 다크 ↔ 화이트가 전환되고, 선택이 브라우저에 저장되어 다음 실행 때도 유지됩니다.

**팁·주의사항**

- 테마는 8.4의 "설정 저장하기"에 포함시켜 백업·복원할 수 있습니다.
- 어두운 교실에서 빔프로젝터로 시연할 때는 화이트 모드가 판서처럼 잘 보입니다.

## 부록: 전체 단축키 한눈에 보기

아래 표는 각 장 끝에 정리된 단축키를 한곳에 모은 것입니다. 더 자세한 조작 절차는 괄호로 표시한 원래 장을 참고하세요.

### A. 전체 도구·화면·편집 (1장 기준)


**공통 규칙**

- 글자 입력 칸(텍스트 편집, 검색창 등)에 커서가 있는 동안에는 아래 단축키가 동작하지 않습니다.
- 도구 선택 단축키는 `Ctrl`/`Alt`와 함께 누르면 무시됩니다(브라우저 단축키 보호).

**도구 선택**

| 키 | 도구 |
|---|---|
| `V` | 선택 |
| `R` | 회전 |
| `E` | 자르기(가위) |
| `S` | 사각형 |
| `O` | 타원 |
| `Y` | 직각삼각형 |
| `L` | 직선 |
| `P` | 꺾은선 |
| `C` | 곡선 |
| `D` | 자유 그리기 |
| `T` | 텍스트 |
| `Shift+T` | 라벨러(㉠ 지시선) |
| `F` | 함수 입력 (단, 삼각형 선택 중엔 삼각형 상하 반전으로 동작) |
| `N` | 점 |
| `A` | 각도 호 |
| `Shift+A` | 직각 표시 |
| `Tab` | (각도 호/직각 도구 사용 중) 두 도구 상호 전환 |
| — | 자·각도기는 버튼 전용 (단축키 없음) |

**그리기 중 모디파이어**

| 상황 | 키 | 효과 |
|---|---|---|
| 사각형·타원 드래그 중 | `Shift` | 정사각형/정원 |
| 직선 그리기 중 | `Ctrl` | 각도 스냅 |
| 꺾은선·곡선 그리기 중 | `Enter` 또는 더블클릭 | 완성 |
| 꺾은선·곡선 그리기 중 | `Esc` | 취소 |
| 자르기 절단선 긋는 중 | `Shift` / `Shift+Ctrl` | 직선 절단 / 각도 스냅 절단 |

**선택·편집**

| 키 | 동작 |
|---|---|
| 클릭 / `Shift`+클릭 | 선택 / 다중 선택 |
| 빈 곳 드래그 | 범위 선택 |
| 더블클릭 | 그룹 멤버 지목 · 텍스트/수식/라벨 편집 · 좌표평면 설정 |
| `Delete` | 선택 오브젝트(또는 선택된 안내선) 삭제 |
| `Ctrl+C` / `Ctrl+V` | 복사 / 마우스 위치에 붙여넣기 |
| `Shift+C` / `Shift+V` | 스타일·각도 복사 / 적용 |
| `G` / `Shift+G` | 그룹 / 그룹 해제 |
| `K` | 잠금 토글 |
| `F` | 선택된 삼각형 상하 반전 |
| 화살표 / `Ctrl`+화살표 | 0.5mm / 5mm 이동 |
| `Shift+↑` / `Shift+↓` | 선 굵기 +0.1 / −0.1mm |
| `PageUp` / `PageDown` | 앞으로 / 뒤로 (회전 도구에선 ±5° 회전) |

**화면·보기**

| 키·조작 | 동작 |
|---|---|
| 휠 / `Shift`+휠 | 상하 / 좌우 이동 |
| `Ctrl`+휠 | 확대·축소 |
| `Space`+드래그, 휠 버튼 드래그 | 자유 이동 |
| `Alt+Enter` | 전체화면 |

**파일·검색·기타**

| 키 | 동작 |
|---|---|
| `Ctrl+S` | 프로젝트 저장(JSON 파일) |
| `Alt+P` | 이미지로 내보내기 창 열기 |
| `Ctrl+F` | 오브젝트 검색 |
| `Ctrl+Shift+F` | 기출문항 검색 |
| `Ctrl+T` | 이미지 객체화 창 열기 |

| `Ctrl+Z` / `Ctrl+Shift+Z` | 실행 취소 / 다시 실행 |

> **길을 잃었을 때**: 어떤 도구를 켰는지 헷갈리면 하단 바 오른쪽의 한 줄 안내를 보세요. 지금 켜진 도구 이름과 조작법이 항상 표시됩니다. 도형을 하나 그리면 자동으로 선택 도구(V)로 돌아온다는 것만 기억하면 대부분의 혼란이 풀립니다.
