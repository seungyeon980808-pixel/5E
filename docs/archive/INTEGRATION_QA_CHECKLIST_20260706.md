# 통합 검증 체크리스트 — v0.50.0 (integration-hub)

작성: 2026-07-06 KST · 대상: `integration-hub` (HEAD `1e8ac1b`) · 저장 스키마: 0.16

이 문서는 **허브 병합(work-dev · cut-tool-fixes · image-dev · equation-font-latinmodern)** 으로
합쳐진 각 브랜치의 기능이 통합 후에도 정상인지 사람이 반복 검증하기 위한 세트다.
기존 앱 회귀(라벨러·각도호·글꼴정책·선택영역 export·로마숫자)는 **별도 문서
`QA_VISUAL_CHECKLIST_20260701.md`(A~G)** 로 검증하고, 이 문서는 §F에서 그 실행만 참조한다.

## 사용 방법

1. `5E_hub\run-server.bat` 실행(포트 8100) → 브라우저에서 앱을 연다. (8097은 세션 검증 서버가 쓸 수 있음)
2. 콘솔(F12)을 열어둔 채 아래 각 항목을 순서대로 확인하고 **실제 결과 / Pass·Fail / 비고** 를 채운다.
3. 회귀 의심 시 캔버스 화면과 내보내기 결과(PNG·SVG)를 함께 비교하고, 어긋난 항목 ID를 비고에 남긴다.
4. **범례**: ✅=사전 정적 검증 완료(이 세션) · ⬜=브라우저에서 확인 필요 · ⚠=알려진 이슈/백로그(비고 참고).

> 병합으로 들어온 기능만 다룬다. 각 브랜치가 무엇을 넣었는지는 섹션 제목에 표기.
