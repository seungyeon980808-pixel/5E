# 도구 UX 개편 확인 체크리스트 (20260706)

작성: 2026-07-06 · 대상 버전: v0.50.7 · 브랜치: `feat/tool-ux-revamp`
참고 문서: `docs/BACKLOG_TOOL_UX_20260706.md` (8건 결정 원문)

이 문서는 이번 배치로 바뀐 8건을 **사람이 직접 브라우저에서** 눈으로 확인하기 위한 체크리스트다.
자동화 스모크 테스트(콘솔 이벤트 dispatch)는 이미 통과했지만, 실제 클릭/드래그 감각과 화면
표시는 직접 눌러보는 게 가장 확실하다.

## 실행 준비
1. 터미널에서 서버 실행:
   ```
   cd C:\Users\user\Desktop\project\51_5E\5E_toolux_dev
   python -m http.server 8791
   ```
2. 브라우저에서 `http://localhost:8791` 접속.
3. 상단에 `[BRANCH: integration-hub]` 경고 배너가 보이는 건 무시해도 된다(이번 작업과 무관한
   기존 하드코딩, 별건으로 확인 예정).

## ① 라벨러 입력 = 텍스트 도구와 동일 (코드 변경 없음 — 기존 동작 재확인용)
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| 라벨러 생성 시 편집창 | 좌측 "라벨러" 버튼(Shift+T) 클릭 → 캔버스에서 두 번 클릭(지시선 시작→끝) | 텍스트 도구와 똑같이 생긴 편집창이 뜬다("라벨 텍스트 입력" 제목, 미리보기+글씨체/크기+빠른문자 버튼) | |
| 더블클릭 재편집 | 만든 라벨러를 더블클릭 | 같은 편집창이 기존 텍스트를 채운 채로 다시 열림 | |
| 저장 확인 | 텍스트 입력 후 확인(또는 Ctrl+Enter) | 라벨 내용이 캔버스에 반영, 지시선 위치(p1/p2)는 그대로 | |

## ② 자유그리기 기본값
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| 기본 채움/선 | "자유그리기" 버튼 클릭 → 캔버스에서 대충 닫힌 도형 모양으로 드래그 | 완성된 도형이 **흰색 채움 + 옅은 회색(연한) 테두리**로 보임(이전엔 테두리 없이 흰 채움만) | |
| 인스펙터 값 | 방금 만든 도형 선택 → 우측 인스펙터에서 선/채움 확인 | 선 굵기 0.2mm, 선 밝기(레벨) 흰색 계열로 표시 | |

## ③ Shift+↑/↓로 선 굵기 조절
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| 굵기 증가 | 선 있는 도형(사각형/직선 등) 선택 → Shift+↑ 여러 번 | 선이 점점 굵어짐(인스펙터 선 굵기 값도 0.1mm씩 증가) | |
| 굵기 감소 + 최소값 | Shift+↓ 여러 번 눌러 0까지 내려보기 | 0 밑으로는 안 내려감(음수 안 됨) | |
| Undo 1번에 원복 | Shift+↑를 3~4번 연속으로 누른 뒤 Ctrl+Z **한 번만** | 늘어난 굵기가 한 번에 원래 값으로 복귀(누른 횟수만큼 여러 번 Undo할 필요 없음) | |

## ④ 각도 도구 통합 (A / Shift+A / Tab)
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| A = 각도 호 | 캔버스에 포커스 있는 상태에서 A 키 | "각도 호" 도구가 활성화(좌측 버튼 파란색 강조) | |
| Shift+A = 직각 표시 | Shift+A 키 | "직각 표시" 도구로 전환 | |
| Tab으로 토글 | 위 상태에서 Tab 키를 눌렀다 뗐다 반복 | 각도 호 ↔ 직각 표시가 계속 왕복 전환(포커스가 다른 곳으로 안 튐) | |
| Shift+G 무반응 확인 | 아무 도형 선택 후 Shift+G | 예전처럼 직각 표시가 뜨지 않음(그룹 해제 기능만 남아있어야 함 — 그룹된 오브젝트가 있다면 그룹 해제만 동작) | |

## ⑤ 자르기 단축키 E
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| E로 자르기 활성화 | E 키 | "자르기(가위·칼·올가미)" 도구 활성화 | |
| K는 더 이상 자르기 아님 | 먼저 V(선택)로 돌아온 뒤 K 키 | 아무 도구도 안 바뀜(선택 도구 유지) | |

## ⑥⑦⑧ 좌측 패널 재편 (함수 입력 공통 이동+F / 좌표축·좌표평면 버튼 삭제)
| 확인 항목 | 하는 법 | 기대 결과 | Pass/Fail |
|---|---|---|---|
| "함수" 카테고리 소멸 | 좌측 패널 스크롤해서 카테고리 목록 확인 | "함수"라는 별도 섹션이 없음(회로/전자기학/광학/역학만 있음) | |
| 함수 입력이 공통 도구에 | "공통 도구" 섹션(V/S/O/... 있는 곳) 확인 | "함수 입력" 버튼이 여기 포함돼 있음, 툴팁에 (F) 표시 | |
| F 단축키로 함수 입력 | F 키 | "함수 입력" 모달이 뜸(수식 입력창) | |
| 함수 입력 시 좌표평면 자동 생성 | 아무것도 선택 안 한 상태에서 F → `sin(x)` 그대로 "객체로 삽입" | 좌표평면(격자+축)이 자동으로 생기고 그 위에 그래프가 그려짐 | |
| 좌표축 생성 버튼 없음 | 공통 도구 섹션 전체 확인 | "좌표축" 버튼이 안 보임(X 단축키도 무반응) | |
| 좌표평면 단독 생성 버튼 없음 | 좌측 패널 전체 확인 | "좌표평면"만 단독으로 만드는 버튼이 안 보임(함수 입력을 통해서만 간접 생성됨) | |
| 자유그리기는 버튼 전용 | F 키를 눌렀는데 자유그리기가 아니라 함수 입력이 뜨는지 재확인, 자유그리기는 버튼 클릭으로만 활성화되는지 확인 | F는 함수 입력, 자유그리기 버튼은 클릭해야만 켜짐(단축키 없음) | |
