# 시각 QA 체크리스트 (20260701)

작성: 2026-07-01 KST · 대상 버전: v0.36.1 (G섹션 v0.36.3) · 저장 스키마: 0.15

이 문서는 **라벨러·각도호·글꼴 정책·선택영역 내보내기**가 최근/향후 변경으로 깨지는지를
사람이 반복 검증하기 위한 수동 QA 세트다. 코드를 수정하지 않으며, 검증 절차와 기대 결과만 정의한다.

## 사용 방법

1. 로컬 서버 실행 후 앱을 연다 (`docs/qa-fixtures/README.md` 참고).
2. 픽스처 `docs/qa-fixtures/label_text_export_fixture.json`을 **파일 → 열기**로 불러온다.
   (불러오기 흐름이 막히면 `label_text_export_expected.md`의 수동 재현 단계를 따른다.)
3. 아래 표의 각 항목을 순서대로 확인하고 **실제 결과 / Pass·Fail / 비고**를 채운다.
4. 회귀 의심 시 캔버스 화면과 export 결과(PNG·SVG)를 함께 비교한다.

> 권장 브라우저: Chrome 또는 Edge (File System Access 저장 picker 지원).
> 글꼴은 export에 임베드되지 않으므로, 내보낸 기기의 설치 글꼴에 따라 모양이 달라질 수 있다(정상 동작).
