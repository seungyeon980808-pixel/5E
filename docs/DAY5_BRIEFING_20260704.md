### 1-1. 숨은 기능 노출 — 범위 미확정 (9건 백로그)
지난 조사에서 11건 발견 → 2건은 이미 처리(기본값 모달 안내 문구 / 그룹화 UI는 이미 있었음 확인).
**나머지 9건은 우선순위를 못 정했음**(질문에 응답 없어 최소 범위로 진행함):

| 우선순위(체감) | 기능 | 현재 상태 |
|---|---|---|
| 중 | Ctrl+F 오브젝트 검색 모달 | 완전히 구현됨, 트리거 버튼/힌트 전혀 없음 |
| 낮 | Alt+P 내보내기 단축키 | 동작하지만 메뉴 툴팁에 표기 안 됨 |
| 낮 | Shift+C/Shift+V (각도 복사/붙여넣기) | 완전히 구현됨, UI 힌트 없음 |
| 낮 | K (잠금 토글) | 인스펙터에 체크박스로 이미 있음, 단축키만 미표기 |
| 낮 | PageUp/PageDown (z-order / 회전 모드에선 5도씩 회전) | 힌트 없음 |
| 낮 | 방향키 (넛지, Ctrl+방향키 큰 이동, 회전모드에선 반전) | 힌트 없음 |
| 낮 | F2 (텍스트 편집) | 더블클릭/우클릭으로 이미 발견 가능, F2는 덤 |
| 낮 | 텍스트 자동변환(theta/lambda/roman1 등 타이핑 시 기호 치환) | 팔레트 버튼으로 같은 결과를 얻는 경로 있음, 다만 설명 없이 자동 치환되는 게 당황스러울 수 있음 |

**판단 필요**: 이 중 뭘 지금 하고 뭘 버릴지(또는 전부 스킵) 결정 필요. 개인적으로는
Ctrl+F 검색 아이콘 하나 추가하는 것 정도만 임팩트 대비 비용이 낮다고 봄. 나머지는
사용 중 불편함이 실제로 느껴질 때 처리해도 될 정도로 낮은 우선순위.
